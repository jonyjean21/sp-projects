# TeamRide

## ステータス: active

## 概要
少年サッカーチームの送迎（ライドシェア）を管理するモバイルWebアプリ。

## 目的・背景
- 保護者の送迎負担を軽減
- 誰がどの試合に送迎するかを可視化
- チーム内のコミュニケーションを円滑化

## 技術スタック
- HTML / CSS / JavaScript（バニラ）
- モバイルファースト・レスポンシブ
- デザイン: Material Design風、グリーンアクセント（#2e7d32）

## 機能一覧

| 機能 | 説明 | 優先度 | 状態 |
|------|------|--------|------|
| ボトムナビゲーション | ページ切り替えUI | 高 | 完了 |
| アクティビティカード | 活動情報のカード表示 | 高 | 完了 |
| スティッキーヘッダー | 固定ヘッダー | 中 | 完了 |
| 送迎マッチング | 保護者と試合の紐付け | 高 | 未着手 |
| 通知機能 | 送迎リマインダー | 中 | 未着手 |

## 画面・UI
- シングルページアプリ（`index.html`）
- ボトムナビゲーションで画面切り替え
- カードベースのレイアウト

## 制約・ルール
- バニラJS（フレームワーク不使用）
- モバイル最優先のレスポンシブ設計
- GitHub Pages でホスティング可能な構成
