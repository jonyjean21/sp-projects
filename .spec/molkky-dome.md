# モルック・ドーム（Maru Village）

## ステータス: active

## 概要
兵庫県川西市を拠点とするモルックコミュニティ。独自の代替経済（マルタ）とデジタルツール群で運営。

## 目的・背景
- モルックを軸としたコミュニティ運営の効率化
- 週次チャプチェ会（定例会議）の議事録自動化
- 複数のアプリ・ツールの開発管理

## 技術スタック
- Notion（中央管理）
- GitHub（marutamura org）
- Manus（プロトタイピング）
- Vercel（デプロイ）
- Firebase（リアルタイムデータ）

## 運営構成
- メンバー: 約30名、5部門体制
- 代替経済: 「マルタ」（木製ディスクトークン、日本円非交換）
- 定例: 週次チャプチェ会

## 外部連携
- **Firebase**: チャプチェキュー（`/chapche-queue`）
- **Notion API**: 議事録投稿
- **Google Drive**: 会議録音テキスト保存
- **GAS**: 自動監視スクリプト

## 自動化パイプライン
```
Google Meet → Gemini文字起こし → Drive保存
→ GAS検知 → Firebaseキュー → Claude Code
→ 議事録生成 → Notion投稿 + ローカル保存 + git push
```

## 制約・ルール
- チャプチェ会の議事録フォーマットは固定（テンプレート準拠）
- メンバーの個人情報は公開リポジトリに含めない
- マルタ経済の詳細は内部限定
