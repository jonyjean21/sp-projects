3ヶ月のポイントレース → 上位12名の頂上決戦
毎月の合戦・定期大会の成績をポイント化。 3ヶ月（1シーズン）のポイント上位12名だけが出場できる招待制シーズン大会を開催。 合戦や定期大会では味わえない、本気のバトルの舞台。

対象: 川西モルックの会 所属メンバー限定
サイクル: 3ヶ月制（3ヶ月ポイントレース → シーズン大会 / 年4回）
開催場所: 満願寺（雨天時は予備日に延期）
集計: 専用アプリで管理・自動集計
開催条件: ポイント上位12名中、10名以上の参加で大会を開催

合戦は規模が大きくガチ度が高いため高配点。ただし定期大会3つ全勝なら月12pt → 合戦優勝(10pt)すら超える。コツコツ参加する人も、合戦で勝つ人も、両方報われる設計。

予選落ち = -1pt で出場するだけではポイントが伸びない。勝ち上がってこそ価値がある。

3ヶ月の短いサイクルで年4回チャンスあり。シーズンごとにリセットされるので逆転も起きやすく、モチベーション維持に最適。
初回は2026年3月スタート → 6月に第1回シーズン大会を開催。
開催日は各シーズン開始時にあらかじめ確定させ、参加者が予定を立てやすくする。

開催条件
10名以上の参加で大会を開催
ポイント上位12名に出場権を付与し、10名以上が参加表明した場合にシーズン大会を開催する。
12名参加（フル開催）
通常フォーマット（3ブロック × 4名 → 決勝トーナメント）で実施。
10〜11名参加
参加者で大会を実施。不参加者はポイント順でそのまま下位に確定する。
例: 10名参加の場合、不参加の2名は11位・12位となる。
10名未満
大会は開催せず、ポイントランキング順がそのままシーズン順位として確定する。

大会を実施しない場合でもポイントレースは有効。3ヶ月間のポイントは記録され、シーズン順位として確定する。
合戦FINAL出場権はシーズン大会を実施した場合のみ優勝者に付与。

試合形式
オールタイマン（1対1）
全試合が1対1の真剣勝負。チーム戦なし。
ステージ形式
予選ラウンド2先（2本先取 / 最大3ゲーム）
決勝トーナメント3先（3本先取 / 最大5ゲーム）

ステージが上がるほどガチ度UP。予選はテンポ良く、決勝Tはじっくり戦える設計。

集計システム
専用アプリで自動集計
各大会の結果を入力すると、ポイントを自動計算してランキングをリアルタイム更新。
シーズン中のランキング変動をいつでも確認できる。

機能: 大会結果入力 / ポイント自動計算 / リアルタイムランキング / シーズン別履歴
技術: GitHub Pages + Firebase RTDB で構築予定

合戦FINAL（既存ルール）
FINAL 年間を通じた実績者による頂上決戦
毎月のモルック合戦で決勝に進出した者に加え、シーズン大会の優勝者も出場権を獲得。
1年間を通じて結果を残してきた精鋭が、翌年1月に一堂に会して最強を決める。

出場資格: 以下のいずれかを満たす選手
① 3月〜12月のモルック合戦で決勝に進出
② シーズン大会で優勝（年4回チャンスあり）
開催時期: 翌年1月
開催場所: 満願寺（雨天時は予備日に延期）
