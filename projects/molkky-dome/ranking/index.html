PART 1

川西モルックの会
最強は誰だ！？シーズン大会

コンセプト
3ヶ月のポイントレース → 上位12名の頂上決戦
毎月の合戦・定期大会の成績をポイント化。 3ヶ月（1シーズン）のポイント上位12名だけが出場できる招待制シーズン大会を開催。 合戦や定期大会では味わえない、本気のバトルの舞台。

対象: 川西モルックの会 所属メンバー限定
サイクル: 3ヶ月制（3ヶ月ポイントレース → シーズン大会 / 年4回）
開催場所: 満願寺（雨天時は予備日に延期）
集計: 専用アプリで管理・自動集計

ポイント制度

REGULAR 定期大会（各月1回 / 〜20名程度参加）
モルックナイト / 平日カップ / 早朝カップ
順位ポイント
優勝4 pt
準優勝3 pt
3位2 pt
準決勝進出1 pt
予選落ち-1 pt

合戦は規模が大きくガチ度が高いため高配点。ただし定期大会3つ全勝なら月12pt → 合戦優勝(10pt)すら超える。コツコツ参加する人も、合戦で勝つ人も、両方報われる設計。

予選落ち = -1pt で出場するだけではポイントが伸びない。勝ち上がってこそ価値がある。

試合形式

全試合が1対1の真剣勝負。チーム戦なし。
ステージ形式

集計システム
専用アプリで自動集計
各大会の結果を入力すると、ポイントを自動計算してランキングをリアルタイム更新。
シーズン中のランキング変動をいつでも確認できる。

機能: 大会結果入力 / ポイント自動計算 / リアルタイムランキング / シーズン別履歴
技術: GitHub Pages + Firebase RTDB で構築予定

PART 2

新人年間王決定戦

コンセプト
新人の中で最強は誰だ？ 1年の集大成
毎月の新人戦で結果を残した選手が年末に集結。 1月〜12月の新人戦 優勝者・準優勝者による個人戦で、 新人の中の年間最強を決める。

目的: 新人の年間を通じた参加モチベーション維持 + 新人にもスポットライトを
開催時期: 毎年12月

出場資格
👑

年間新人王決定戦

1月〜12月の新人戦 優勝者 / 準優勝者による個人戦

毎月の新人戦で優勝・準優勝を勝ち取った選手が年末に集結。 新人の中で最強は誰なのか、1年の総決算として決着をつける。
  • 優 各月 新人戦 優勝者（最大12名）
  • 準 各月 新人戦 準優勝者（最大12名）

最大24名が出場権を獲得（重複出場者がいれば実人数は減少）

大会フォーマット
個人戦 新人年間王決定戦
1
予選リーグ（ブロック分け）
出場者を4名ずつのブロックに分割。各ブロック内で総当たり戦。
2
各ブロック上位者が決勝トーナメント進出
参加人数に応じて通過枠を調整。シード権は年間成績上位者に付与。
3
決勝トーナメント
1対1のタイマン勝負。準決勝・決勝は3本先取で白熱の展開に。
年間スケジュール
1月〜12月 — 毎月の新人戦で優勝・準優勝 = 出場権獲得
　6月: 中間発表（出場権獲得者一覧を公開）
　11月: 最終確定（出場者リスト・トーナメント表作成）
12月 — 新人年間王決定戦 開催
翌年1月 — 合戦FINAL 開催（合戦決勝進出者 + シーズン大会優勝者 + 新人年間王が出場）
年間の全体像

＋
新人年間王決定戦 × 年1回（12月）
＋

↓
川西モルックの会の1年を彩る大会体系

シーズン大会で日頃の実力を可視化し、新人年間王で新人にスポットライトを。 12月の新人年間王 → 翌年1月の合戦FINALで、年間のクライマックスを締めくくる。

合戦FINAL（既存ルール）
FINAL 年間を通じた実績者による頂上決戦
毎月のモルック合戦で決勝に進出した者に加え、シーズン大会の優勝者と新人年間王も出場権を獲得。
1年間を通じて結果を残してきた精鋭が、翌年1月に一堂に会して最強を決める。

出場資格: 以下のいずれかを満たす選手
① 3月〜12月のモルック合戦で決勝に進出
② シーズン大会で優勝（年4回チャンスあり）
③ 新人年間王決定戦で優勝
開催時期: 翌年1月
開催場所: 満願寺（雨天時は予備日に延期）
