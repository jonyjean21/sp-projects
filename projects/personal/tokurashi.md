## SEO方針（チーム決定）

- バックデートは禁止（Google公式の推奨に従う）
- 一括公開はOK（Google公式見解）
- 品質で勝負、週1〜2本ペースで定期更新
- リライトは定期的に実施（AI臭除去 + 画像挿入）

## WP認証

| 項目 | 内容 |
|------|------|
| WP_USER | TOKURASHI_WP_USER（.envに記載） |
| WP_APP_PASSWORD | TOKURASHI_WP_APP_PASSWORD（.envに記載） |
| WAF | SiteGuard（PUTブロック / 大画像403 / 連続POST制限） |

WAF対策: mediumサイズ画像を使用、更新の間隔を空ける。

## 関連ファイル（sp-projectsリポジトリ）

| ファイル | 役割 |
|---------|------|
| `tools/tokurashi-gen.py` | 記事生成スクリプト（Gemini+Pexels+WP全自動） |
| `tools/tokurashi-rewrite.py` | リライトスクリプト（AI臭除去+画像挿入） |

## 記事生成コマンド

```bash
# 記事を生成して公開
python3 tools/tokurashi-gen.py --topic "節約術" --publish

# リライト実行
python3 tools/tokurashi-rewrite.py
```

環境変数はすべて `.env` に記載済み。
