# MOLKKY HUB

> モルックを愛するすべてのひとのために

## 概要

モルックの総合情報メディアサイト。ルール解説・大会情報・チーム紹介・ツール紹介など、日本のモルック情報を一元的に集約する「ハブ」を目指して運営中。

## サイト構成

- **ルール・遊び方** — 初心者向けの基本ルール解説、ゲームの流れ・得点計算・勝敗の決まり方
- **大会情報** — 全国のモルック大会・練習会の情報掲載
- **チーム紹介** — 全国のモルックチームのプロフィールページ
- **記事一覧** — モルックの人気・魅力に関する記事コンテンツ（寄稿含む）
- **ツール** — スコア管理アプリまとめ等
- **正規品ガイド** — TACTIC社製モルック全ラインナップの紹介
- **YouTubeチャンネル一覧** — モルック関連YouTuberのディレクトリ
- **NICE MOLKKY JAPAN** — 提携団体のページ（大会詳細・レギュレーション・ギャラリー等）

## 寄稿プロジェクト

### 概要

外部の寄稿者からの記事を掲載するプロジェクト。サイトのコンテンツ量拡大に大きく貢献。

### 現状

| 項目 | 内容 |
|------|------|
| レギュラー連載者 | 3名 |
| 不定期寄稿 | NICE MOLKKY JAPAN |
| 更新頻度 | 寄稿含め週2〜3回 |
| フォーマット | 定型フォーマットあり。基本的に寄せられた内容をそのまま掲載 |
| 連絡手段 | 寄稿者ごとにバラバラ |

### 課題・改善ポイント

- 寄稿者が今後増えそう → 受付・管理フローの整備が必要
- 連絡手段がバラバラ → 統一 or 集約の仕組みがほしい
- 入稿〜公開のフローを効率化できる余地あり

## 運営上の課題と方針

### 最大のボトルネック

**PC作業時間が限られている** → すべての施策は「少ない時間で回せるか」を基準に判断する

### 取り組み方針

以下は検討中の方向性。今後の深掘りで優先度・具体策を詰めていく。

### 1. 大会情報パイプラインの進化

**現状（稼働中）:**
```
手動で情報収集 → Google スプレッドシートに入力 → GAS → WordPress 自動投稿
```

**次のステップ:**
- 情報収集の部分を効率化 or 自動化できないか検討
- X API / IFTTT / RSS / 手動+半自動のいずれかを選定
- X API Free プランは制約大 → 費用対効果を見て判断

### 2. 記事投稿の効率化

**現状:**
- 未着手

**検討中のアプローチ:**
- 定型記事（チーム紹介・大会レポ等）のテンプレート + AI下書き
- 寄稿プロジェクトの受付〜公開フローの効率化
- GAS / WP REST API を活用した投稿の半自動化

### 3. SEO改善

**現状:**
- 検索流入が下落傾向
- リライト後の再評価期間の可能性あり

**検討事項:**
- Search Consoleのデータ確認（クエリ別の順位変動）
- 主要ページのSEO最適化
- コンテンツ拡充による検索カバレッジ拡大

### 4. 収益化

**検討中のアプローチ:**
- スポンサー獲得（メディアキット作成 → 営業）
- アフィリエイト（モルック用品紹介）
- サイト価値を上げてから本格着手

### 5. マルタ村との連携

- モルハブ側との連携強化（詳細は `projects/molkky-dome/` を参照）
