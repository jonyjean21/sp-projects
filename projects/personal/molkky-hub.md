# MOLKKY HUB

> モルックを愛するすべてのひとのために

## 概要

モルックの総合情報メディアサイト。ルール解説・大会情報・チーム紹介・ツール紹介など、日本のモルック情報を一元的に集約する「ハブ」を目指して運営中。

## 基本情報

| 項目 | 内容 |
|------|------|
| URL | https://molkky-hub.com |
| ステータス | 運用中 |
| 運営 | 個人（管理人：孔雀） |
| 技術スタック | WordPress + SWELL テーマ |

## サイト構成

- **ルール・遊び方** — 初心者向けの基本ルール解説、ゲームの流れ・得点計算・勝敗の決まり方
- **大会情報** — 全国のモルック大会・練習会の情報掲載
- **チーム紹介** — 全国のモルックチームのプロフィールページ
- **記事一覧** — モルックの人気・魅力に関する記事コンテンツ
- **ツール** — スコア管理アプリまとめ等
- **正規品ガイド** — TACTIC社製モルック全ラインナップの紹介
- **YouTubeチャンネル一覧** — モルック関連YouTuberのディレクトリ
- **NICE MOLKKY JAPAN** — 自主大会の運営（大会詳細・レギュレーション・ギャラリー等）

## 今後の取り組み

### 大会情報掲載の自動化
X（Twitter）のタイムラインから大会・練習会情報を収集し、Googleスプレッドシートに反映。GASでWordPressに自動投稿する仕組みを構築する。

### 記事投稿の半自動化
記事作成・投稿ワークフローの効率化。

### スポンサー獲得に向けた施策
サイトの収益化・スポンサーシップの確立。
