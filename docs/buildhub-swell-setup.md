# BuildHub SWELL 初期設定ガイド

WP管理画面 → 外観 → カスタマイズ の順に設定。

---

## 1. サイト基本情報
**外観 → カスタマイズ → サイト基本情報**
- サイトのタイトル: `BuildHub`
- キャッチフレーズ: `Claude Code・AI開発ツールの最新情報を毎日お届け`
- サイトアイコン（ファビコン）: お好みで設定

---

## 2. ヘッダー設定
**SWELL設定 → ヘッダー**
- ヘッダーレイアウト: `center`（中央揃え）または `left`
- ロゴ: テキストのまま「BuildHub」でOK（後で画像ロゴに変更可）
- ヘッダー背景色: `#1a1a2e`（ダーク紺）または `#0073aa`（WPブルー）
- ヘッダー文字色: `#ffffff`

---

## 3. カラー設定
**外観 → カスタマイズ → カラー**
- メインカラー: `#0073aa`（テック系ブルー）
- テキストリンク色: `#0073aa`
- ボタンカラー: `#0073aa`

---

## 4. トップページ設定
**設定 → 表示設定**
- ホームページの表示: `固定ページ`
- ホームページ: `トップ`
- 投稿ページ: `（なし）`

---

## 5. メニュー設定
**外観 → メニュー → 新しいメニューを作成**

メニュー名: `グローバルナビ`
追加するページ:
- トップ（ホーム）
- Claude Code まとめ（カテゴリ）
- このサイトについて（Aboutページ）

メニューの位置: `グローバルナビゲーション`にチェック

---

## 6. フッター設定
**SWELL設定 → フッター**

フッターテキスト:
```
© 2026 BuildHub | Claude Code・AI開発ツール情報メディア
<a href="/privacy-policy/">プライバシーポリシー</a> | <a href="/contact/">お問い合わせ</a>
```

---

## 7. 投稿設定
**SWELL設定 → 投稿・固定ページ**
- アイキャッチ画像: 表示する
- 著者情報: 非表示（BuildHub編集部として運営のため）
- 目次: 自動表示（h2が3つ以上の時）

---

## 8. SNS設定（将来用）
**SWELL設定 → SNS・OGP**
- OGPの設定: 有効
- Twitterカードタイプ: `summary_large_image`
- OGPデフォルト画像: サイトのOGP用画像をアップロード（1200×630px推奨）

---

## 9. アドセンス枠（将来用）
収益化フェーズに入ったら以下を設定:
- **SWELL設定 → 広告コード** → 記事上・記事下・サイドバーにアドセンスコードを入力

---

- ヘッダーメニューが機能している
