# 手順書②: Google Alerts 設定（大会情報の自動収集 Phase 1）

> 所要時間: 約10分 / 費用: 無料

## やること

「モルック 大会」等のキーワードでGoogle Alertsを設定し、ウェブ上の新着大会情報を自動キャッチする。

## 手順

### Step 1: Google Alerts にアクセス（1分）

1. https://www.google.co.jp/alerts にアクセス（Googleログイン状態で）

### Step 2: アラートを作成（5分）

以下の3つのキーワードを1つずつ登録する。

#### アラート①: 「モルック 大会」

1. 検索窓に 「モルック 大会」 と入力
2. 「オプションを表示」 をクリック
3. 以下のように設定:
   - 頻度: その都度
   - ソース: 自動
   - 言語: 日本語
   - 地域: 日本
   - 件数: すべての結果
   - 配信先: RSS フィード
4. 「アラートを作成」 をクリック
5. 作成されたアラートの横に RSSアイコン が表示される → 右クリック → リンクのアドレスをコピー
6. このURLをメモしておく（後でスプシに使う）

#### アラート②: 「モルック 参加者募集」

- 同じ手順で作成。配信先は RSS フィード
- RSSのURLをメモ

#### アラート③: 「モルック 開催」

- 同じ手順で作成。配信先は RSS フィード
- RSSのURLをメモ

### Step 3: スプレッドシートでRSSを取り込む（5分）

1. 大会情報用のスプレッドシートを開く
2. 新しいシートを追加して 「Google Alerts」 と名前をつける
3. A1セルに以下の数式を入力（URLはStep 2でコピーしたものに差し替え）:

```
=IMPORTFEED("ここにアラート①のRSS URL", "items", FALSE, 20)
```

4. B列以降に、アラート②、③も同様に:

```
=IMPORTFEED("ここにアラート②のRSS URL", "items", FALSE, 20)
```

5. 自動的にタイトル・URL・日付が表示される

### 別のやり方: 1シートに全部まとめる

A1に以下の形式で入力すると、各列の意味が分かりやすい:

| セル | 数式 |
|------|------|
| A1 | `=IMPORTFEED("RSS URL", "items title")` |
| B1 | `=IMPORTFEED("RSS URL", "items url")` |
| C1 | `=IMPORTFEED("RSS URL", "items created")` |

## 完了後

- Google がウェブ上で「モルック 大会」に関する新しいページを見つけるたびにRSSに追加
- スプレッドシートが自動更新される（IMPORTFEED はスプシが開かれるたびに更新）
- 大会情報の「漏れ」を減らせる

## 注意点

- Google Alerts は X（Twitter）の投稿は拾えない → Phase 2 の IFTTT で補完
- リアルタイム性は低い（数時間〜1日のラグ）→ 速報性は手動チェックで補完
- IMPORTFEED はスプシを開いていないと更新されない → GASで定期更新も可能（Phase 3）
