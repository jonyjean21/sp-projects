# 手順書①: GA4 自動エクスポート設定

> 所要時間: 約30分 / 費用: 無料

## やること

GA4のデータを毎月1日にスプレッドシートへ自動出力する。

## 手順

### Step 1: GA4 プロパティIDを確認（3分）

1. https://analytics.google.com/ にログイン
2. 左下の 歯車アイコン（管理） をクリック
3. プロパティ設定 を開く
4. プロパティID（数字のみ、例: `123456789`）をメモ

### Step 2: スプレッドシートでApps Scriptを開く（2分）

1. https://docs.google.com/spreadsheets/d/1z9bmgRvXAlWFLoARZSM7mAiosrEuy7jkSF6jh82T1So/ を開く
2. 上部メニュー → 拡張機能 → Apps Script
3. 新しいタブでスクリプトエディタが開く

### Step 3: GA4 Data API を有効化（5分）

1. スクリプトエディタの左メニューで 「サービス」の横にある「＋」 をクリック
2. 一覧から 「Google Analytics Data API」 を探してクリック
3. 「追加」 をクリック
4. 左メニューの「サービス」に `AnalyticsData` が追加されればOK

### Step 4: スクリプトを貼り付け（5分）

1. エディタの既存コード（`function myFunction() {}` 等）を 全て削除
2. このリポジトリの `scripts/ga4-monthly-export.gs` の中身を 全て コピーして貼り付け
3. 先頭付近の `const GA4_PROPERTY_ID = 'XXXXXXXXX';` を、Step 1で確認したプロパティIDに書き換え
   ```javascript
   const GA4_PROPERTY_ID = '123456789'; // ← 自分のIDに変更
   ```
4. Ctrl + S で保存

### Step 5: 初回実行 & 権限承認（5分）

1. 上部のプルダウンで `testExportCurrentMonth` を選択
2. ▶ 実行 をクリック
3. 「承認が必要です」と表示される → 「権限を確認」 をクリック
4. Googleアカウントを選択
5. 「このアプリは確認されていません」→ 「詳細」 → 「（安全でないページ）に移動」
6. 「許可」 をクリック
7. スプレッドシートに戻って 「月次サマリー」「チャネル別」「ページ別」 の3つのシートにデータが入っていれば成功

### Step 6: 月次トリガーを設定（3分）

1. スクリプトエディタに戻る
2. 上部のプルダウンで `setupTrigger` を選択
3. ▶ 実行 をクリック
4. 実行ログに「トリガー設定完了」と表示されればOK
5. これで毎月1日 朝7時に自動実行される

### Step 7: 動作確認（2分）

1. スクリプトエディタ左メニュー → トリガー（時計アイコン）
2. `monthlyExport` のトリガーが表示されていればOK

## 完了後

- 毎月1日に前月のデータがスプシに自動追記される
- チャットで「先月のデータ見て」と言ってもらえれば分析する
- スプシのURLを共有してくれれば直接見れる（要権限設定）

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| 「AnalyticsData is not defined」 | GA4 Data APIが追加されていない | Step 3をやり直す |
| 「権限がありません」 | OAuth承認が通っていない | Step 5をやり直す |
| データが0 | プロパティIDが間違っている | Step 1で正しいIDを確認 |
| シートが作成されない | スプシのURLが違う | スクリプトが正しいスプシに紐づいているか確認 |
