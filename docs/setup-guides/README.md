# セットアップ手順書

PCを開いたときに、順番に実行するだけで設定が完了するガイド集。

## 実行順序

| # | 手順書 | 所要時間 | 費用 | 何が変わるか |
|---|--------|---------|------|------------|
| 1 | [GA4 自動エクスポート](01-ga4-export-setup.md) | 30分 | 無料 | 毎月のGAデータがスプシに自動出力される |
| 2 | [Google Alerts 設定](02-google-alerts-setup.md) | 10分 | 無料 | ウェブ上の大会情報が自動でキャッチされる |
| 3 | [IFTTT Pro 設定](03-ifttt-setup.md) | 30分 | $2.99/月 | X上のモルック投稿がスプシに自動記録される |
| 4 | [SEO クイックウィン](04-seo-quickwin-guide.md) | 30〜40分 | 無料 | 検索流入の改善が始まる |

## 合計

- **所要時間:** 約1.5〜2時間（1日でやっても、分割してもOK）
- **月額費用:** $2.99（約450円）のみ
- **効果:** データ確認の自動化 + 情報収集の自動化 + SEO改善の第一歩
