# 手順書④: SEOクイックウィン実施ガイド

> 所要時間: 約30〜40分 / 費用: 無料

## やること

Search Console で「表示されてるのにクリックされてない」キーワードを見つけて、タイトルとmeta descriptionを改善する。これが最も工数対効果が高いSEO施策。

## 手順

### Step 1: Search Console でお宝キーワードを探す（15分）

1. https://search.google.com/search-console にログイン
2. 左メニュー → 「検索パフォーマンス」 → 「検索結果」
3. 日付フィルタを 「過去3か月」 に変更
4. 「平均CTR」 と 「平均掲載順位」 にチェックを入れる（デフォルトでは非表示）
5. 下のクエリ一覧が表示される

### Step 2: 改善対象を特定（5分）

以下の条件に当てはまるクエリが「お宝キーワード」:

| 条件 | 意味 |
|------|------|
| 表示回数 100以上 | 十分に検索されている |
| CTR 5%以下 | 表示されてるのにクリックされてない |
| 掲載順位 3〜15位 | 改善すれば1ページ目の上位に入れる |

クエリ一覧を「表示回数」の降順で並べ替え → 上から見ていって上記条件に合うものをピックアップ。

### Step 3: 該当ページを確認（3分）

1. 見つけたクエリをクリック
2. 「ページ」 タブに切り替え
3. そのクエリで表示されているページのURLが分かる

### Step 4: タイトル & meta description を改善（15分）

1. WordPressでそのページの編集画面を開く
2. ページ下部にスクロール → 「SEO SIMPLE PACK」の設定欄がある
3. 以下のルールで改善:

#### タイトル改善のコツ

| 改善前 | 改善後 | ポイント |
|--------|--------|---------|
| モルックのルール | 【完全ガイド】モルックのルール｜得点・勝敗・反則まで解説 | キーワード前置き + 具体性 |
| 大会情報 | モルック大会情報 2026年【全国版】開催スケジュール一覧 | 年号 + 「全国」で検索意図カバー |
| チーム紹介 | 全国のモルックチーム一覧｜地域別に探せる | 「一覧」「探せる」で利便性アピール |

タイトルの基本形: `【修飾語】メインキーワード｜補足説明`

#### meta description 改善のコツ

- 120〜160文字（PCで全文表示される長さ）
- 検索キーワードを冒頭に含める（太字で表示されてクリック率UP）
- ユーザーが得られるものを具体的に書く
- 「～をまとめました」「～が分かります」で締める

例:
```
モルックのルールを初心者向けに完全解説。得点計算・勝敗の決まり方・反則ルールまで、
これ1ページで全てが分かります。図解付きで、初めてプレイする方でもすぐに理解できます。
```

### Step 5: 結果を記録（2分）

改善したページを記録しておく（次月のSearch Consoleで効果確認するため）:

| ページ | 対象クエリ | 改善前CTR | 改善日 | 変更内容 |
|--------|-----------|----------|--------|---------|
| /rules/ | モルック ルール | 3.2% | 2026-02-XX | タイトルに【完全ガイド】追加 |

→ ここで記録した内容をチャットで教えてもらえれば、来月の効果測定を一緒にやれる。

## 改善後の注意

- 効果が出るまで 2〜4週間 かかる
- 焦って再変更しない。4週間は待つ
- Search Console > URL検査 > 「インデックス登録をリクエスト」で再クロールを促進できる

## 次回（来月）やること

- 改善したページのCTR変化を確認
- 新たなお宝キーワードがないかチェック
- → 月次ルーチンに組み込む
