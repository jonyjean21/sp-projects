# 川西モルックの会 ランキング制度 & ガチ大会 企画書

> 発案: 歩さんとの会話より / 2026年2月

---

## コンセプト

**「川西モルックの会で一番強いのは誰だ？」**

定期大会の成績をポイント化し、シーズンを通じて安定的に強いメンバーを可視化する。
シーズン上位12名だけが出場できる**招待制ガチ大会**を開催し、
普段の合戦や定期大会では味わえない本気のバトルを実現する。

### 狙い

- 定期大会へのモチベーション向上（毎回のポイントが意味を持つ）
- 安定的に強いメンバーの可視化（一発勝負ではなくシーズン通算）
- 合戦や定期大会とは別格の「ガチの舞台」の創出
- 川西モルックの会の一体感・ブランド強化

---

## 対象・参加資格

- **川西モルックの会 所属メンバー限定**
- ガチ大会への出場はシーズンランキング上位12名のみ

## ポイント制度

### 対象大会とポイント配分

#### モルック合戦（月1回）

メインイベント。トーナメント形式のため順位に応じた高配点。

| 順位 | ポイント |
|------|---------|
| 優勝 | 10 pt |
| 準優勝 | 7 pt |
| 3位 | 5 pt |
| ベスト4 | 4 pt |
| ベスト8 | 3 pt |
| ベスト16 | 2 pt |

#### 定期大会（モルックナイト・平日カップ・早朝カップ）

各月1回開催。合戦より軽めの配点だが、積み重ねで差がつく。

| 順位 | ポイント |
|------|---------|
| 優勝 | 3 pt |
| 準優勝 | 2 pt |
| 3位 | 1 pt |

### ポイント獲得の考え方

- モルック合戦は規模が大きくガチ度が高いため、配点を厚くしている
- 定期大会は参加しやすい反面、配点は控えめ → 合戦の結果が大きな差に
- ただし定期大会を全勝すれば月9pt → 合戦準優勝(7pt)を上回る
- **コツコツ出場する人も、合戦で勝つ人も、両方報われる設計**

### シーズン期間

| 方式 | 期間 | 対象大会数の目安 |
|------|------|-----------------|
| 3ヶ月制 | 例: 4月〜6月 / 7月〜9月 / 10月〜12月 / 1月〜3月 | 合戦3回 + 定期大会9回 = 12大会 |
| 半年制 | 例: 4月〜9月 / 10月〜3月 | 合戦6回 + 定期大会18回 = 24大会 |

> **推奨**: まずは半年制でスタートし、データが溜まったら3ヶ月制への移行を検討

## ガチ大会 フォーマット

### 出場資格

シーズンランキング **上位12名**

### 大会形式: 全タイマン（1対1）

#### 予選ラウンド（3ブロック × 4名）

- 12名を3ブロック（A・B・C）に振り分け
- 各ブロック内で総当たり（各選手3試合）
- **各ブロック上位2名（計6名）** → 無条件で決勝トーナメント進出

#### ワイルドカード決定戦（残り2枠）

- 各ブロック3位の3名から2名が敗者復活
- **検定対決**などの特別ルールで決定（例: モルック検定の成績、特殊ルール戦など）
- ドームらしい遊び心のある選考方法で盛り上げる

#### 決勝トーナメント（8名）

```text
        【準々決勝】         【準決勝】          【決勝】
    ┌─────────┐
    │ A1 vs WC2│
    └────┬────┘
         ├──────┐
    ┌────┴────┐  │
    │ B2 vs C2 │  │
    └─────────┘  │
                  ├──────┐
    ┌─────────┐  │      │
    │ B1 vs WC1│  │      │
    └────┬────┘  │      │
         ├──────┘      │
    ┌────┴────┐         │
    │ A2 vs C1 │         │
    └─────────┘         │
                         │
                      【優勝】
```

> 組み合わせはシード順（ランキング上位が有利な配置）

### 試合形式

| 方式 | 説明 | 所要時間目安 |
|------|------|-------------|
| **3先（3本先取）** | 先に3ゲーム勝った方が勝利。最大5ゲーム | 長め・ガチ度MAX |
| **2先（2本先取）** | 先に2ゲーム勝った方が勝利。最大3ゲーム | コンパクト・テンポ良い |

> **推奨**: 予選は2先、決勝トーナメントは3先（ステージが上がるほどガチ度UP）

```text
【ポイントレース期間】4月〜9月（6ヶ月）

  4月  合戦 + ナイト + 平日 + 早朝
  5月  合戦 + ナイト + 平日 + 早朝
  6月  合戦 + ナイト + 平日 + 早朝  ← 中間ランキング発表
  7月  合戦 + ナイト + 平日 + 早朝
  8月  合戦 + ナイト + 平日 + 早朝
  9月  合戦 + ナイト + 平日 + 早朝  ← 最終ランキング確定

【ガチ大会】10月（予選 + 決勝トーナメント）

【シーズン2 ポイントレース】10月〜3月
【シーズン2 ガチ大会】4月
```

- [ ] ガチ大会の会場・日程の決め方
- [ ] 同ポイントの場合の順位決定方法

### 同ポイント時のタイブレーク案

1. 直接対決の成績
2. モルック合戦の最高順位
3. 大会出場回数（多い方が上位）
4. 検定対決

### 12傑ランキングとの違い

12傑ランキングは別企画（ドーム全体のランキング制度）。
本企画は**川西モルックの会メンバー限定**の「ガチ度」に特化したもの。

- 12傑 = ドーム全体の功績・実力ランキング
- 本企画 = 川西モルックの会の定期大会ポイントレース → 招待制ガチ大会
