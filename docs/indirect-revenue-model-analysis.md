# 間接収益モデル分析 — SPビジネスへの適用

> 参考: 金沢容「間接収益モデル」記事（2026年3月）
> 作成日: 2026-03-03

## 記事の核心

### 間接収益モデルとは
- 顧客から直接お金をもらわない
- 代わりに顧客の「情報」や「顧客の顧客」を収益化する
- 顧客を「情報の源泉」「ハブ（中継地点）」として捉え直す

### キーフレーズ
> 「その顧客が持っている、まだ誰も手をつけていない資産」を探し、そこから自分までどうやってサプライチェーンを引っ張るかを考える

### 記事の実例
- **美容室**: 無料の予約システムを提供 → 美容室の顧客（美意識の高い女性）に化粧品を売る
- **学習塾**: 無料の教材ツール → 学生にPC・賃貸・資格講座を売る
- **クリニック**: 医療隣接の事業者にクリニックを提供 → 対価として事業者の顧客を丸ごともらう（Win-Win）

## SPビジネスへの適用分析

### 1. MOLKKY HUB（最も直接的に適用可能）

**現状**: アフィリエイト月3,000円 / 月3,000セッション

**顧客をハブとして捉え直す:**

| 顧客 | 未開拓の資産 | サプライチェーンの引き方 |
|---|---|---|
| 大会主催者 | 参加者リスト・会場ネットワーク・地域行政コネ | 無料の大会管理ツール提供 → 参加者へスポンサー商品配信 |
| チームリーダー | メンバー（アウトドア好き・休日アクティブ層） | 無料チーム管理アプリ → メンバーへアウトドア用品レコメンド |
| モルック用品メーカー | 購入者データ・販売チャネル | HUBで無料掲載 → 購入者への関連商品クロスセル |

**ポイント**: 日本唯一のモルック専門メディア = 情報独占ポジション。顧客情報の資産価値が極めて高い。

### 2. マルタ村 / モルックドーム（BtoBtoCの宝庫）

**既に持っている資産:**
- 10個以上のアプリ（スタンプカード、ファンコミュニティ、ギルドシステム等）
- コミュニティの行動データ
- 物理的な拠点（モルックドーム）

**網目状の収益網:**

```
スタンプカードアプリ（無料）
  → 来店データ = 「誰が何回来たか」の情報資産
    → 常連層にだけ見える特別オファー（地域飲食店・宿泊施設）
    → 地域事業者から逆営業が来る構造

ファンコミュニティアプリ（無料）
  → コミュニティの属性データ = ハブ
    → 「ファミリー層 × アウトドア好き × 兵庫在住」
    → この属性に合うスポンサーを接続
```

**クリニックモデルの直接応用:**
- モルックドームを地域事業者に無料/低コストで貸し出す
- 対価 = お金ではなく「事業者の顧客にモルック体験を提供する機会」
- 例: 子供会・PTA → 無料体験会 → 親子がリピーター化
- 例: 企業の福利厚生 → 無料法人利用 → 社員が個人リピーター化

### 3. 20収入源計画への追加

| 現在のモデル（直接収益） | 間接収益への転換案 |
|---|---|
| モルックHUBアフィリエイト | 無料大会運営ツール → 参加者へのスポンサー商品紹介 |
| マイクロSaaS（有料） | 無料化 → 利用者データを元にした企業マッチング |
| コンサル（直接課金） | 無料AI導入支援 → 導入先の顧客ネットワーク活用 |

## 最重要の問い

> **「モルックHUBの月3,000人の訪問者、マルタ村のコミュニティメンバーが持っていて、まだ誰も手をつけていない資産は何か？」**

この問いの答えが見つかれば、「一本のルート」が「網目状の収益網」に進化する。

---

## 本質

**ハブを磨く → 資産が溜まる → 売る。**

商売は「一本のルート」から「網目状に広がる巨大な収益網」へと進化する。
