# マルタ村 全体設計

## マルタ村とは

マルタ村（兵庫県川西市）を拠点とした **コミュニティ経済圏** の構想。
モルックドームという物理的な拠点を核に、デジタルサービス群で
「人・モノ・体験・文化」をつなぐ。

## ビジョン（要すり合わせ）

> マルタ村に関わる人が、自然と助け合い、楽しみ合い、経済が回る仕組みを作る

**キーワード候補:**
- コミュニティ経済圏（マルタ経済圏）
- 性善説文化
- ハッピー文化
- 地域 × デジタル

## アプリ間の接続マップ

```
                    ┌──────────┐
                    │ LINE Bot │ ← ユーザーの入口（日常）
                    └────┬─────┘
                         │ メッセージ
                    ┌────▼─────┐
                    │Claude AI │ ← 意図判定・Notion操作・Issue作成
                    └────┬─────┘
                         │
              ┌──────────┼──────────┐
              ▼          ▼          ▼
         ┌────────┐ ┌────────┐ ┌────────┐
         │ Notion │ │ GitHub │ │各アプリ │
         └────────┘ └────┬───┘ └───┬────┘
                         │         │
                    レビュー    SSO連携
                    サイクル    (実装中)
```

### 既に繋がっている線
- LINE Bot → Notion（コンテンツ管理）
- LINE Bot → GitHub（Issue自動作成）
- oshi-katsu → SSO（sso_token検証）

### まだ繋がっていない線（候補）
- 各アプリ → 共通SSO認証
- 各アプリ → LINE Bot（通知配信）
- モルハブ → ドーム系アプリ（相互送客）
- モノハブ → あちらさまからです（モノの流通 → お礼/感謝）

## 全体設計で決めるべきこと（要議論）

### A. 優先度: 高

| # | 論点 | 選択肢 | 現状 |
|---|------|--------|------|
| A1 | **マルタ村のポータル（入口）は何か？** | LINE Bot / 専用Webアプリ / 既存アプリの拡張 | LINE Botが事実上の入口 |
| A2 | **SSO基盤をどこまで広げるか？** | 全アプリ必須 / 主要アプリのみ / 不要 | oshi-katsuのみ |
| A3 | **ユーザーDBは統合するか？** | 共通DB / アプリ個別のまま / SSO連携のみ | アプリ個別 |

### B. 優先度: 中

| # | 論点 | 選択肢 | 現状 |
|---|------|--------|------|
| B1 | **マルタ経済圏（ポイント/通貨）は作るか？** | ポイント制度 / ギフトカード / なし | Notion上に構想あり |
| B2 | **アプリの統廃合は必要か？** | 全部維持 / 機能重複を統合 / 実験的なものは棚上げ | 10+アプリが並走 |
| B3 | **モルハブとの連携はどう実現するか？** | APIレベル / リンク相互掲載 / コンテンツ連携 | 検討中 |

### C. 優先度: 低（将来）

| # | 論点 |
|---|------|
| C1 | マルタ村ポータルアプリの開発 |
| C2 | マルタギフトカード制度のデジタル化 |
| C3 | コエダ町との連携構想 |
