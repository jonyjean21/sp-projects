1. 画面の見方

画面下のナビゲーションバーで各ページに切り替えます。

2. パスワードについて

「記録」と「投稿」のページはパスワード保護されています。初回アクセス時にパスワード入力が求められます。

パスワードは gosaki です。一度入力すればブラウザを閉じるまで再入力は不要です。

設定ページでの選手編集・追加時にもパスワードが必要です（同じパスワード）。

3. 試合を記録する

基本的な試合記録の流れです。

  1. 下のナビから ✏️ 記録 をタップ
    パスワード入力画面が出たら「gosaki」を入力
  2. 日付を確認・変更する
    今日の日付が自動で入ります。過去の試合を入れる場合は日付を変更してください
  3. 挑戦者をドロップダウンから選択
    選手の役割（王者/門番/挑戦者）がタグで表示されます
  4. 防衛側をドロップダウンから選択
    門番や王者を選ぶと「試合タイプ」が自動で切り替わります（「自動検出」バッジが出ます）
  5. スコアを入力（挑戦者 / 防衛側それぞれ）
    0〜15から選択。勝者側のスコアが黄色でハイライトされます
  6. 黄色の 「記録する」 ボタンをタップ
    「○試合を記録しました」のトースト通知が出たら成功です
挑戦者・防衛側のどちらかを選んでいない場合や、スコアが両方0の場合は「記録する」ボタンが押せません。

4. 複数試合をまとめて記録する

同じ日に複数試合があった場合、まとめて記録できます。

  1. 1試合目を入力した後、「+ もう1試合追加」 ボタンをタップ
    新しい試合入力フォームが下に追加されます。日付は前の試合から自動で引き継がれます
  2. 2試合目の 挑戦者・防衛側・スコア を入力
  3. 必要な試合数だけ追加を繰り返す
    何試合でも追加できます。各試合の右上の「✕」で個別に削除もできます
  4. 全て入力したら 「記録する」 をタップ
    追加した全ての試合が一括で保存されます
日付を変えれば別の日の試合も同時に登録できます。各試合で個別に日付を設定してください。

5. 記録を修正する

入力ミスがあった場合の修正方法です。

  1. 下のナビから 📋 全試合 をタップ
  2. 修正したい試合を見つける
    上部のフィルター（「全て」「門番戦」「タイトルマッチ」や選手名）で絞り込めます
  3. 試合の右端にある ✎（鉛筆）ボタン をタップ
    パスワード入力を求められる場合があります
  4. 編集モーダルで内容を修正する
    修正できる項目: 日付、挑戦者、防衛側、試合タイプ、スコア（両方）
  5. 「保存」 ボタンをタップ
    「修正しました」のトースト通知が出たら成功です
スコアを変更すると勝者も自動で変わります。スコアが同点の場合はエラーになりませんが、正しい結果になっているか確認してください。

6. 記録を削除する

誤って記録した試合を削除する方法です。

  1. 上記の修正と同じ手順で ✎ ボタン から編集モーダルを開く
  2. モーダル下部の赤い 「削除」 ボタンをタップ
  3. 確認ダイアログで OK をタップ
    削除は取り消せないので注意してください
一度削除した試合は元に戻せません。間違い防止のため、削除前にスコアを確認してください。

7. 新しい選手を追加する

初めて参加する選手を登録する方法です。2つの方法があります。

方法A: 設定ページから追加

  1. 下のナビから ⚙️ 設定 をタップ
  2. 「+ 新しい選手を追加」 ボタンをタップ
    パスワード入力が求められます
  3. 名前 と Xアカウント（任意）を入力
    Xアカウントは「@xxx」の形式で入力。なければ空欄でOK
  4. 「追加」 ボタンをタップ

方法B: 試合記録中に追加

  1. 記録ページで挑戦者 or 防衛側のドロップダウンの一番下にある 「+ 新しい選手」 を選択
  2. 同じ追加モーダルが開くので、名前とXアカウントを入力
  3. 追加後、ドロップダウンから新しい選手を選び直す
新しい選手は自動的に「挑戦者」として登録されます。門番への昇格は設定ページの選手編集から変更できます。

8. 選手情報を編集する

名前の修正、Xアカウントの追加、役割の変更ができます。

  1. 下のナビから ⚙️ 設定 をタップ
  2. 編集したい選手の行を タップ
    パスワード入力が求められます
  3. 編集モーダルで変更したい項目を修正
    変更できる項目: 名前、Xアカウント、役割（王者/門番/元王者/挑戦者）
  4. 「保存」 ボタンをタップ
名前を変更すると過去の試合記録も自動で更新されます。正しい名前を入力してください。

選手を削除する場合は、編集モーダルの赤い「削除」ボタンをタップします。

9. X投稿を作成する

記録した試合からX投稿用テキストを自動生成できます。

  1. 下のナビから 📤 投稿 をタップ
    パスワード入力が求められます
  2. 投稿したい試合を チェックボックスで選択
    日付ヘッダーをタップするとその日の全試合を一括選択できます
  3. 上部の黄色い 「投稿を生成」 ボタンをタップ
  4. テンプレートタブからお好みの形式を選択
    用途別テンプレート: 🏆結果報告 / 💥試合速報 / 💪防衛報告 / 🎉新王誕生 / ✏️カスタム / 📋全試合結果 / 🛡️門番戦歴 / 👑タイトルマッチ
  5. 「X投稿をコピー」 ボタンをタップしてクリップボードにコピー
  6. Xアプリに切り替えて ペースト して投稿
    文字数は右下に表示されます。280文字を超えると赤くなります
テンプレートの使い分け
🏆
結果報告 — 定番。試合結果をフォーマルに報告
💥
試合速報 — カジュアルに試合結果を速報
💪
防衛報告 — 門番が全勝した週のまとめ向き
🎉
新王誕生 — チャンピオン交代時の特別テンプレート
✏️
カスタム — シンプルなスコア一覧。自由に編集向き

10. 困ったときは

Q. データが消えた？

データはスマホのブラウザ（localStorage）に保存されています。ブラウザのキャッシュを削除するとデータが消えます。別のブラウザ（Chrome/Safari切り替えなど）でも別データになります。常に同じブラウザでアクセスしてください。

Q. パスワードを忘れた

パスワードは gosaki です。

Q. スコアを間違えて記録した

「📋 全試合」ページから ✎ ボタンで修正できます。→ 5. 記録を修正する

Q. 選手が見つからない

新しい選手は先に登録が必要です。→ 7. 新しい選手を追加する

Q. 「記録する」ボタンが押せない

以下を確認してください:

  • 挑戦者と防衛側を両方選んでいるか
  • 挑戦者と防衛側が同じ選手になっていないか
  • スコアが両方0になっていないか（最低どちらか1以上必要）

Q. X投稿が280文字を超える

「✏️ カスタム」テンプレートはシンプルなので文字数が少なくなります。または投稿する試合数を減らしてみてください。

Q. 別のスマホからも見たい

データは各端末のブラウザに個別に保存されているため、端末間でデータは共有されません。1台の端末で管理することをおすすめします。
