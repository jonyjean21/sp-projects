## 核心の問い：われわれは何を目指すのか

### 結論

> **MOLKKY HUBは「日本モルックのScorehlio」になる。**
>
> メディア（情報発信）だけではなく、**ツール（スコア管理・大会運営・チーム管理）を無料で提供し、モルック活動の「インフラ」になる**。
> そのインフラの上に、間接収益モデルを構築する。

### なぜメディアだけではダメなのか

| | jajapatatas（競合） | MOLKKY HUB（われわれ） |
|---|---|---|
| アプローチ | 情報メディア（はてなブログ） | **インフラ+メディア** |
| 収益モデル | アフィリエイト/広告 | **間接収益（ツール→資産化→多角的収益）** |
| 技術の天井 | はてなブログの制約内 | **制約なし（自前開発）** |
| 防衛壁 | 先行者優位+人脈 | **ツールのスイッチングコスト** |
| スケーラビリティ | 線形（記事を書いた分だけ） | **指数的（ユーザーが増えるほどデータが溜まる）** |

jajapatatasは「情報を集めて発信する」モデル。われわれは「ツールを提供してデータを集める」モデル。
**勝負する土俵が違う。**

## 戦略の3本柱

### 柱1: 「なくてはならないツール」を作る

**最優先で開発すべき3つのツール：**

#### (A) スコア管理・戦績トラッキング（= Strava for モルック）
- リアルタイムスコア記録（既存の「5先王」を発展）
- 個人・チームの戦績蓄積
- 対戦成績、勝率、スキットル命中率等の分析
- **なぜ最優先か**: 競技のたびに使う「必需品」になる。使うたびにデータが溜まり、離脱コストが上がる

#### (B) 大会検索・登録・管理（= Scoreholio for モルック）
- 全国の大会をワンストップで検索（地域・日程・規模でフィルタ）
- オンラインエントリー・決済
- 大会主催者向けの運営ツール（ブラケット自動生成、スコアボード）
- **なぜ最優先か**: 現在最大のペインポイント「大会情報が散在して見つけにくい」を解消。主催者と参加者の両方を取り込める

#### (C) チーム・仲間マッチング
- 「近くのチーム/練習会を探す」インタラクティブマップ
- メンバー募集掲示板
- 初心者→チームへの導線
- **なぜ重要か**: 362万人の体験者 → 5万人のアクティブ転換率1.4%を上げる最大のレバー

### 柱2: メディアで「入口」を作る（既存資産の強化）

現在のWordPress + MOLKKY HUBサイトを活かして：
- **初心者ガイド**の充実（jajapatatasの弱み）
- **用具レビュー・比較記事**（アフィリエイトの起点にもなる）
- **大会レポート**（自動化パイプラインを活用）
- **SEO強化**（Google検索流入が-12.3%の回復）

メディアの役割は「ツールへの導線」。記事で興味を持った人を → ツール利用者に転換する。

### 柱3: コミュニティの「ハブ」として信頼を蓄積

- ツールを無料で提供し、感謝と信頼を蓄積
- 大会主催者・地域団体との関係構築
- ユーザーの声を聞き、改善し続ける
- **JMAとの関係**: 競合ではなく補完。協会がカバーしきれない領域を埋める

## 成功の条件（リサーチから抽出）

### 1. 「レジストレーション・アンカー」を押さえる

> スポーツテック業界では、**選手登録機能が最もスティッキー**。
> 登録プラットフォームを押さえた事業者が、隣接サービスに展開して「ウォレットシェア」を拡大するのが主流パターン。

MOLKKY HUBでも「選手・チーム登録」を軸に据え、そこからスコア管理、大会管理、EC、配信へ展開する。

### 2. 「Value Creation Before Value Extraction」

> コミュニティ主導型ビジネスの最大のリスクは「過度に攻撃的なマネタイズでコミュニティ自体を毀損すること」。

Phase 0〜1では徹底的に無料で価値提供。収益化は信頼の蓄積の上に乗せる。

### 3. 「大手が来ない」ニッチの防衛壁

> ニッチ市場では「その分野のオンリーワン」になることが最重要。市場が小さいほどシェアを取りやすく、一度「公式プラットフォーム」的な地位を確立すると、競合参入の動機が薄い（市場が小さすぎるため大手が来ない）。

モルック市場はVCが投資するには小さすぎるが、個人/少人数チームが「ハブ」を取るには十分大きい。

### 4. コミュニティ投資のROI

- 強いコミュニティを持つ企業は、持たない企業の **2.1倍の速度で収益成長**
- 顧客生涯価値（LTV）が **46%高い**
- 顧客獲得コスト（CAC）が **26%低い**
- コミュニティへの投資1ドルあたり、平均 **6.4ドルのリターン**

## 競合との戦い方

### vs jajapatatas（全国モルックカレンダー）

**正面衝突しない。土俵を変える。**

| jajapatatasの領域 | MOLKKY HUBの領域 |
|---|---|
| 大会「情報」の集約 | 大会「運営」のツール化 |
| 週末イベントまとめ記事 | イベント検索・フィルター・通知 |
| 独自ポイントシステム（手動） | 自動スコア集計・戦績蓄積 |
| 選手インタビュー | 選手プロフィール・実績DB |
| はてなブログ上のコンテンツ | Webアプリとしてのツール群 |

jajapatatasが「メディア」として優れているなら、われわれは「ツール」で勝つ。
**情報を「読む」場所ではなく、モルックを「やる」ための場所になる。**

### vs JMA（日本モルック協会）

**競合ではなく補完。**

JMAがカバーしきれない領域（非公式大会の情報集約、デジタルスコア管理、チームマッチング）をMOLKKY HUBが埋める。
JMAの公式大会の運営ツールとして採用されることが理想的なゴール。

## リスクと対策

| リスク | 対策 |
|--------|------|
| 開発リソースの不足 | Claude Code + AI活用で1人でも開発可能な範囲に絞る |
| ユーザーが集まらない | まず大会主催者5〜10人を巻き込み、そこから参加者に広げる |
| jajapatatasの対抗 | 土俵を変えているので直接競合しない。むしろ情報面で協力できる |
| JMAとの関係悪化 | 協会の公式ルール・方針を尊重。補完的なポジションを明確に |
| 収益化に時間がかかる | Phase 0〜1は投資と割り切る。本業収入でカバー |
