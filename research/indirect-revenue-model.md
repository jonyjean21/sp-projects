# 間接収益モデル・ハブ磨き戦略 リサーチ

調査日: 2026-03-03

## 1. グローバルSaaSの成功事例

### Notion（コミュニティ駆動型フリーミアム）
- ユーザー1億人超、有料400万以上
- 売上: 2024年に4億ドル（2019年は300万ドル）
- 無料→有料転換率: 約13%
- ユーザーが自発的にテンプレ共有→マーケ費ゼロで拡散

### Figma（コミュニティ・レッド・グロース）
- 9,000万ユーザー、売上8.2億ドル
- マーケットプレイス手数料15%
- 新規Enterprise顧客の70%は元Professionalプラン

### Strava（スポーツコミュニティのハブ）
- 登録1.2億人、MAU 5,000万
- 売上2.75億ドル（90%がサブスク）
- 有料転換率わずか2%だが年額79.99ドルの高単価
- スポンサー付きチャレンジ、匿名データ販売も

### Discord（無料コミュニティから間接収益）
- 売上5.75億ドル
- 基本無料、有料は「自己表現とコミュニティ貢献」に特化
- Server Boost: 「仲間のために課金する」心理

## 2. ニッチ市場でのパターン

### ディスクゴルフネットワーク（DGN）
- 月額9.99ドルのサブスク配信
- 初年度目標1,500人→**17,000人超**を獲得
- 「TV放映権を追求する必要がなくなった」

### 小規模市場で成り立つモデル
1. まず「なくてはならないツール」になる
2. コミュニティの信頼を蓄積（投資1ドル→平均6.4ドルリターン）
3. 間接収益から始める（アフィリ、決済手数料）
4. 成長とともに直接収益追加
