# 月次マネー振り分けスキル

## 概要
毎月のカード・固定費引落し前に、各口座への振り分け金額を計算して指示書を出す。

## 使い方
- `/mm` — 今月の振り分け指示を作成
- `/mm check` — 各口座の残高 vs 引落し額を確認

## アカウント設定
`data/finance/accounts.json` に口座構造・カード紐付け・引落し日を保存（gitignore済み）

## 処理フロー

### Step 1: 今月の引落し額を確認
ユーザーに聞く or MFスクショから読み取り。
スクショの場合、口座画面の「カード引落し額」タブを見る。

### Step 2: 各口座の現在残高を確認
ユーザーに聞く or MFスクショから読み取り。

### Step 3: 不足額を計算
各引落し口座について:
```
不足額 = 今月の引落し合計 − 現在残高
```
不足の口座だけ振込指示を出す。

### Step 4: 振り分け指示書を出力

```
📋 今月の振り分け指示（YYYY年M月）

【資金源】住信SBI ハイブリッド預金: ¥XXX,XXX

🔴 〜M/2 までに（イオンカード）
  → イオン銀行(駿平) に ¥XX,XXX

🔴 〜M/10 までに（au PAYカード）
  → au じぶん銀行 に ¥XX,XXX

🔴 〜M/25 までに（保育園）
  → 池田泉州銀行 に ¥XX,XXX

🔴 〜M/26 までに（三井住友カード）
  → 住信SBI 代表口座 に ¥XXX,XXX
  → イオン銀行(優依) に ¥XX,XXX

🔴 〜M/27 までに（楽天・PayPay）
  → 楽天銀行(優依) に ¥XX,XXX

🟢 みずほ（住宅ローン）: OK / 要補充

💰 移動後のハイブリッド残高見込み: ¥XXX,XXX
```

### 振り分けルール
1. 資金源は基本「住信SBI ハイブリッド預金」
2. ハイブリッド → 代表口座は SBI内移動（即時・無料）
3. ハイブリッド → 他行は SBI振込（ランクに応じて無料回数あり）
4. 残高が足りる口座はスキップ
5. 保育園(池田泉州)は毎月¥61,000固定

### 注意事項
- data/finance/ は .gitignore済み。絶対にコミットしない
- 金額情報はファイルに残さず、会話内で処理する
- 投資・証券口座は対象外（触らない）
