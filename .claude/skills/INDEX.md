# スキル一覧

> 「何をどのスキルで？」の引き当て用INDEX

## 業務スキル（日常使い）

| コマンド | 用途 | ファイル |
|---------|------|---------|
| `/tournament <URL>` | 大会情報をWPに投稿（URL/キュー/検索/クリーンアップ） | tournament.md |
| `/chapche-minutes` | チャプ会議事録をNotionに自動変換 | chapche-minutes.md |
| `/promote <記事URL>` | 記事からXポスト文を自動生成（UTM付き） | promote.md |
| `/repost` | メンション引用リポスト生成 | repost.md |
| `/naimol-result` | ナイモル大会結果ページ自動生成 | naimol-result.md |
| `/weekly-roundup` | 週間大会まとめ記事の全自動公開（月曜） | weekly-roundup.md |
| `/mm` | 月次マネー振り分け計算 | mm.md |
