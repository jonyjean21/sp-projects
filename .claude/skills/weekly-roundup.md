# /weekly-roundup — 週間大会まとめ記事の自動生成

## 概要
毎週水曜日にその週末（土日）のモルック大会情報をまとめた記事を自動生成し、Xポスト案も出力する。

## トリガー
- セッション開始時のフックで「今日は水曜日」を検知 → 自動実行
- または手動で `/weekly-roundup` を実行

## 実行手順

### Step 1: 今週末の大会データ取得
```python
# WP REST API で tournament CPT を取得
# ACF event_date で今週末（土日）をフィルタ
# 重要: 曜日は必ず calendar モジュールで検証する（手動で曜日を決めない）
```

### Step 2: 各大会の情報整理
- タイトル、場所（都道府県）、主催者
- **リンク先**: ACF の sns_link（X投稿）または detail_link（外部ページ）を使用
- **絶対に架空のURLを使わない** — APIから取得した実在リンクのみ
- sns_link も detail_link もない場合はリンクなし（「情報が入り次第更新」と記載）

### Step 3: 来週以降の注目大会ピックアップ
- 翌週〜2週間先の大会から7件程度ピックアップ

### Step 4: 記事コンテンツ生成
- タイトル: 【今週末のモルック大会】{月/日}・{月/日}の全国大会情報まとめ｜{年}{月}第{N}週
- slug: weekly-molkky-{YYYYMMDD}（週末初日の日付）
- カテゴリ: 週間まとめ(139) + 大会情報(8)
- excerpt: 80字程度の要約
- **曜日は必ず Python calendar で確認**

### Step 5: アイキャッチ画像生成
```bash
python3 tools/eyecatch-gen.py --type weekly \
  --date "{M/D}（{曜}）- {M/D}（{曜}）" \
  --count {件数} \
  --week "{年}年{月}月第{N}週"
```
- 生成した画像をWPメディアにアップロード
- 記事の featured_media に設定

### Step 6: 下書き作成
- status: draft で作成
- SP確認 → 公開

### Step 7: Xポスト案生成
記事公開後、以下のフォーマットでXポスト案を出力:
```
📝 記事更新！【今週末のモルック大会】

{M/D}（{曜}）・{M/D}（{曜}）は全国{N}件のモルック大会が開催🎯

{注目大会1}（{都道府県}）
{注目大会2}（{都道府県}）
{注目大会3}（{都道府県}）
...

詳しくはモルハブでチェック👇
{記事URL}

#モルック #molkky
```

## 重要ルール
1. **曜日は必ず `calendar` モジュールで計算** — 手動判断は禁止
2. **リンクは実在するもののみ** — sns_link / detail_link のAPIデータを使用
3. **tournament CPT の個別ページにはリンクしない** — 参加チーム一覧しか表示されない
4. **アイキャッチは必ず設定** — tools/eyecatch-gen.py で生成
5. **日本時間(JST)で全て処理** — UTC+9
