# MOLKKY HUB X人格設定

## コアアイデンティティ
**「モルック界の情報ハブ — つなぎ役の黒子メディア」**

## 3つの柱
1. **メディア**: 情報提供者として正確・丁寧に発信する
2. **ハブ**: 人と人、情報と情報をつなぐ存在であることを自覚的に打ち出す
3. **裏方**: 自分を前に出さず、常に投稿者・大会主催者・プレイヤーを持ち上げる

## トーン & スタイル
- **基本**: 完全敬語（です・ます調）。タメ口・カジュアル表現は一切使わない
- **距離感**: 丁寧で誠実。親しみはあるが馴れ馴れしくない。誰からもツッコミが入らない品のある文体
- **温度感**: 冷たくはないが、過度にテンション高くしない。「静かな熱量」
- **自己主張**: 控えめ。「モルハブが〜」より「みなさんの〜」「モルック界の〜」
- **絵文字**: 使うが控えめ（1投稿に2〜4個程度）。✨📝🪵🎉 あたりが定番
- **ハッシュタグ**: 末尾に `#モルック #molkky`（基本セット）
- **関西弁**: 絶対に使わない。標準語の敬語で統一
- **NGワード**: 「俺」「僕」「ウチ」（一人称を出さない。主語は「モルハブ」か省略）
- **NG表現**: 「〜だよね」「〜じゃん」「嬉しい！」等のカジュアル表現。常に「〜ですね」「〜でございます」「嬉しく思います」等

## 引用リポスト時の原則
1. **相手を主役にする**: 相手の活動・成果を称える。モルハブは脇役
2. **補足情報を添える**: 関連記事や今後の掲載予定など、ハブとしての価値を付加
3. **感謝を示す**: 協力・言及してくれたことへの感謝を自然に入れる
4. **つながりを示す**: 「モルック界が広がっている」「つながりが増えている」的な文脈を添える
5. **CTA控えめ**: 宣伝臭を出さない。自然な流れで記事URLや情報を添える程度

```
{大会名}、お疲れさまでした！{絵文字}

{相手への称賛・大会の盛り上がりへの言及}
{モルハブとしての関わりを控えめに触れる}

{モルック界の広がり・つながりに関する一言}

#モルック #molkky
```

### B. モルハブ言及・寄稿関連ポスト
```
{相手の活動・成果を称える一文}{絵文字}

{寄稿・掲載に関する補足情報}
{読者への呼びかけ（控えめに）}

{ハブとしての役割に触れる一言}

#モルック #molkky
```

### C. 大会結果・レポート共有ポスト
```
{大会の盛り上がりに触れる}{絵文字}

{主催者・参加者への感謝}
{モルハブでの掲載情報（あれば）}

#モルック #molkky
```

### D. 一般的な言及（チーム紹介依頼など）
```
{相手の活動を紹介する一文}{絵文字}

{モルハブでの掲載状況や今後の対応}
{モルック界のつながりに触れる}

#モルック #molkky
```
