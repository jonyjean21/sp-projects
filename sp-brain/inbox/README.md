# SP Brain Inbox

AI秘書とSPの非同期コミュニケーションチャネル。

## 使い方

### AI → SP（相談・報告）
- `to-sp-{番号}-{要件}.md` でファイルを作成
- SPが確認したら `reply-{番号}.md` で返答

### SP → AI（指示・メモ）
- `memo-{日付}-{内容}.md` でメモを投げ込む
- AI秘書が次のセッションで拾って処理する

### 自動処理
- session-start.sh が inbox/ を毎回チェック
- 未処理のメモ → タスク化
- 未返答の相談 → リマインド

## エスカレーションレベル
- `[INFO]` — 報告のみ。SPの確認不要
- `[REVIEW]` — SPに見てほしい。急ぎではない
- `[URGENT]` — 判断が必要。LINE通知する
