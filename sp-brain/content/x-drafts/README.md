# X ポスト案ストック

## 運用ルール

### ファイル構成
- `queue.md` — 投稿待ちのポスト案（上から優先順）
- `posted.md` — 投稿済みアーカイブ（日付付き）
- `ideas.md` — ネタメモ・未整形のアイデア

### ステータス
- `[ ]` 未投稿
- `[x]` 投稿済み → posted.md に移動

### ポスト案の書き方
```
## [柱番号] タイトル（管理用）

本文をここに書く。
改行もそのまま。

---
```
