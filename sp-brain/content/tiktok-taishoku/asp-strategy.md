## TikTokからの導線設計

```
TikTok動画
  ↓（「プロフのリンクから」で誘導）
プロフィールリンク
  ↓
リンクまとめページ（lit.link or Linktree）
  ├── おすすめ退職代行TOP3（→ ASPリンク）
  ├── 料金比較記事（→ ブログ完成後はここに誘導）

```

4. 📖 退職代行の選び方ガイド → ブログ記事（後日追加）

## SP手動タスク（申請に必要な作業）

### 今すぐやること
1. [ ] TikTokアカウント作成（ビジネスアカウントで）
2. [ ] lit.linkアカウント作成 → プロフィールページ設定
3. [ ] もしもアフィリエイト登録（TikTokをメディアとして登録）
4. [ ] afb登録
5. [ ] アクセストレード登録
6. [ ] A8.netでサイト追加（TikTok or lit.linkページ）

### ASP審査対策
- **サイトURL**: lit.linkのURLを登録（TikTokアカウントURLでもOKなASPあり）
- **もしも**: 審査なしで即提携できる案件が多い
- **A8**: 基本審査あり。既存トクラシアカウントにサイト追加が楽
- **afb**: 審査あり。サイトの記事数が少ないと落ちる可能性 → ブログ立ち上げ後に申請が安全

### 申請の最適順序
1. もしもアフィリエイト（審査なし案件あり → 即日利用可能）
2. A8.net（既存アカウントにサイト追加 → 案件ごとに提携申請）
3. アクセストレード（ガーディアン狙い）
4. afb（ブログ完成後に申請）

## 収益シミュレーション（TikTok経由）

### 保守的シナリオ（月30本投稿）
| 指標 | 数値 |
|------|------|
| 月間投稿数 | 30本 |
| 平均再生数 | 5,000回/本 |
| プロフクリック率 | 1% |
| lit.linkからASPクリック率 | 30% |
| ASP成約率 | 5% |
| 月間成約数 | 30 × 5000 × 0.01 × 0.30 × 0.05 = **2.25件** |
| 単価 | ¥10,000 |
| **月収** | **¥22,500** |

### バズった場合（1本10万再生）
| 指標 | 数値 |
|------|------|
| バズ動画再生数 | 100,000回 |
| プロフクリック率 | 2%（バズ時は高くなる） |
| lit.link → ASPクリック率 | 30% |
| 成約率 | 5% |
| **その1本だけで** | 100,000 × 0.02 × 0.30 × 0.05 = **30件 = ¥300,000** |

→ GW明け（5月第1週）にバズれば一撃数十万の可能性

## 注意事項

1. **ASP規約確認**: TikTokからの直接誘導がOKか、各ASPの規約を確認
2. **ステマ規制**: 動画に「PR」「広告」表示が必要か確認（2023年10月〜）
3. **薬機法等**: 退職代行は薬機法対象外だが、「必ず辞められる」等の断定表現は避ける
4. **複数ASP使い分け**: バズった動画→高単価案件、通常動画→成約しやすい案件
