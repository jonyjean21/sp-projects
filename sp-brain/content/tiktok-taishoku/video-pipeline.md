# TikTok動画 自動生成パイプライン設計

> 作成日: 2026-03-13
> 目的: 退職代行TikTokの動画を最小労力で量産する

---

## 方針: 3段階アプローチ

### Phase 1: Vrew + CapCut（今すぐ開始・推奨）
- **メインツール**: Vrew（https://vrew.app/ja/）— テキスト→動画の自動生成
- **サブツール**: CapCut（デザイン微調整が必要な場合のみ）
- **所要時間**: 1本あたり5-10分（Vrewが大部分を自動化）
- **品質**: 高い（AI音声+自動テロップ+背景動画が一括生成）
- **用途**: 最初の10本はここで作って投稿→反応を見る

### Phase 2: Vrew量産体制（反応確認後）
- **ツール**: ChatGPT/Gemini（台本生成）→ Vrew（動画生成）→ 手動投稿
- **所要時間**: 台本生成1分 + Vrew生成3分 = 1本4-5分
- **品質**: Phase 1と同等
- **用途**: 反応のいいパターンが分かったらテンプレ化して量産

### Phase 3: Python完全自動化（将来・オプション）
- **ツール**: Python + MoviePy + FFmpeg + Gemini
- **所要時間**: スクリプト実行で1本1分以下
- **品質**: 中程度（テロップ+背景+BGMのシンプル構成）
- **用途**: 1日10本以上の大量生産が必要になった場合

## Phase 1: Vrew + CapCut フロー（即実行可能）

### Vrewとは
- **URL**: https://vrew.app/ja/
- **開発**: 韓国Voyager X社（無料プランあり）
- **特徴**: テキストを入力するだけでAI音声+テロップ+背景動画の動画を自動生成
- **日本語対応**: 強い（日本語AI音声が自然）
- **プラットフォーム**: PC（Windows/Mac）+ Web版あり

### Vrewが最適な理由
1. **テキスト→動画が全自動**: 台本を貼るだけでAI音声・テロップ・背景素材を一括生成
2. **顔出し不要に最適**: テロップ動画の制作に特化している
3. **無料枠が十分**: 月間の無料エクスポート枠でスタート可能
4. **CapCutより効率的**: CapCutはテンプレ選択+手動調整が必要だが、Vrewはテキスト入力だけ

### 推奨制作フロー

```
1. スクリプト（scripts.md）からテキストをコピー
2. Vrewを開く → 「テキストから動画を作成」を選択
3. テキストを貼り付け → AI音声・スタイルを選択
4. 自動生成された動画を確認・微調整
5. 9:16（縦型）で書き出し
6. （必要に応じて）CapCutでデザイン微調整
7. TikTokに投稿
```

**1本の制作フロー（5-10分）:**
1. スクリプト選択（1分）
2. Vrewにテキスト貼り付け+AI音声選択（2分）
3. 自動生成を待つ+微調整（3-5分）
4. 書き出し+投稿（2分）

### CapCutの使いどころ（サブツール）
- Vrewで生成した動画にトレンドBGMを追加したい場合
- テロップのデザインを凝りたい場合
- TikTokテンプレートを使いたい場合
- **基本はVrewだけで完結する**

## Phase 2: Vrew量産体制（反応確認後）

### 量産フロー
```
[Step 1] ChatGPT/Geminiに台本テンプレ+テーマを投げる
    → 台本5-10本を一括生成（1分）
    ↓
[Step 2] 各台本をVrewに貼り付け → 動画自動生成（3分/本）
    ↓
[Step 3] 一括書き出し → TikTokに予約投稿風に毎日1本アップ
```

### 台本テンプレ（Gemini/ChatGPTプロンプト）
```
以下のテーマでTikTok動画の台本を作って。
フォーマット:
[0-3秒] フック（感情を揺さぶる一言）
[3-15秒] 共感・情報
[15-25秒] 解決策の提示
[25-30秒] CTA「プロフのリンクから」

テーマ: {業種}の人が退職代行を使うべき理由
ターゲット: 20-30代、{業種}で辞めたいけど言えない人
トーン: 共感→背中を押す
```

### 週間生産目標
- Phase 2では**週10-15本**を目標（Vrew量産で十分可能）
- 毎日2本投稿体制（12:00 + 19:00）

## Phase 3: Python完全自動化パイプライン（将来・オプション）

### 技術スタック

| ツール | 役割 | コスト |
|--------|------|--------|
| Python 3.x | メインスクリプト | ¥0 |
| MoviePy | 動画編集（テロップ挿入、合成） | ¥0 |
| FFmpeg | エンコード | ¥0 |
| Pillow | テロップ画像生成 | ¥0 |
| Gemini API | スクリプト自動生成 | ¥0（無料枠） |
| Pexels API | 背景動画素材 | ¥0 |
| フリーBGM | BGM素材 | ¥0 |

### 処理フロー

```
[Input] スクリプトJSON（テーマ、テロップテキスト、ハッシュタグ）
    ↓
[Step 1] Gemini API → スクリプト自動生成（or JSONから読み込み）
    ↓
[Step 2] Pexels API → テーマに合った背景動画をDL
    ↓
[Step 3] Pillow → テロップ画像を生成（白文字+黒縁取り）
    ↓
[Step 4] MoviePy → 背景動画 + テロップ画像 + BGM を合成
    ↓
[Step 5] FFmpeg → TikTok最適フォーマットで書き出し（9:16, 1080x1920）
    ↓
[Output] mp4ファイル + メタデータ（キャプション、ハッシュタグ）
```

### スクリプト設計案

```python
# tools/tiktok-video-gen.py（将来の実装）
#
# Usage:
#   python3 tools/tiktok-video-gen.py --theme "新卒 退職"
#   python3 tools/tiktok-video-gen.py --script scripts/001.json
#   python3 tools/tiktok-video-gen.py --batch scripts/  # フォルダ内一括
#
# Output:
#   output/tiktok/001_新卒退職.mp4
#   output/tiktok/001_新卒退職_meta.json（キャプション+ハッシュタグ）
```

### TikTok投稿の自動化

**現状**: TikTok公式APIは個人の投稿自動化をサポートしていない
**対策**:
1. 動画は自動生成、投稿はスマホから手動（1分/本）
2. 将来的にTikTok Content Posting APIが使えるようになれば自動化
3. 代替: Instagramリール・YouTube Shortsにもクロスポスト（API経由で自動化可能）

## BGM戦略

### フリーBGM素材サイト
- DOVA-SYNDROME: https://dova-s.jp/ （商用利用OK）
- 甘茶の音楽工房: https://amachamusic.chagasi.com/
- MusMus: https://musmus.main.jp/

### ジャンル別おすすめBGM
| 動画テーマ | BGMジャンル | 雰囲気 |
|-----------|-----------|--------|
| 体験談系 | エモ系ピアノ | 切なさ+希望 |
| 情報系（料金・比較） | ポップ/アップテンポ | 明るく分かりやすく |
| 共感系（パワハラ等） | Lo-Fi/チル系 | 落ち着いた共感 |
| 煽り系（辞めろ） | トレンドBGM | TikTokの流行に乗る |

**注意**: TikTokの商用利用では楽曲の著作権に注意。CapCut内蔵BGMかフリーBGMを使用。
