# AIクローン: Phase 1（記憶層）から着手

- **日付**: 2026-03-09
- **決定**: AIクローンの実装をPhase 1（記憶層 = sp-brain/）から始める

## 背景
- 「もう1人のSP」を作りたい
- いきなりLINE Botを作っても、中身（記憶・判断基準）がないと意味がない
- まず記憶層を整備して、Claude Codeがセッション間で一貫した判断ができるようにする

## フェーズ構成
1. Phase 1: 記憶層（sp-brain/ のMD群）← 今ここ
2. Phase 2: LINE Bot連携（外部からの入力受付）
3. Phase 3: 自律稼働（定期実行、自発的な行動）

## 他の選択肢
- LINE Bot から先に作る → 記憶がないと「ただのAPI中継」になるので却下
- 全部一気に作る → 時間がかかりすぎる。70点で出して改善の方針に合わない
