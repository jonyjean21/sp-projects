# GitHub Pages アプリ群

## 概要
- sp-projects リポジトリで40以上のWebアプリを公開
- 全てバニラ HTML/CSS/JS（フレームワーク不使用）
- ダーク基調 / サイバーパンク風デザインで統一
- appsカタログ（apps/index.html）で一覧管理

## 技術パターン
- フロント: HTML + CSS + バニラJS
- データ: Firebase Realtime DB
- API: Vercel Serverless Functions
- 自動化: GAS（Google Apps Script）

## ルール
- 新アプリ追加時は appsカタログにも追加
- 必ず管理画面（admin UI）を付ける
- commit → push → open を一気に
