## ステップ1: BOOTH登録（20分）

1. https://booth.pm にアクセス
2. 右上の「ログイン」をクリック
3. pixiv IDでログイン（なければ「pixiv IDを作成」→ メアドとパスワードだけで完了）
4. ログイン後、右上メニュー →「ショップ管理」→「ショップ作成」
5. ショップ名: **SP Lab**
6. ショップURL: `sp-lab`（空いてなければ `sp-lab-ai` など）
7. ショップ説明:
   ```
   Claude Code / AI を活用した副業・自動化のノウハウを販売しています。
   WordPress自動化、SEO記事量産、GASスクリプトなど、実際に使って稼いでいるツールとテンプレートを公開中。
   ```
8. 「ショップを作成する」をクリック

### 商品を出品する

9. ショップ管理画面 →「商品管理」→「商品登録」
10. 商品カテゴリ: **同人ソフト・ゲーム > ツール・その他**
11. 商品名:
    ```
    【コピペ即使える】Claude Code 副業プロンプト大全50選 — AIで月10万稼ぐ実践テンプレート集
    ```
12. 商品説明: `income/03-booth-prompts/generated/booth-listing-guide.md` の「商品説明」セクションをコピペ
13. 価格: **¥1,980**
14. 販売形式: **ダウンロード販売**
15. ファイルをアップロード: PDF変換済みファイル
    - PDFがまだなければターミナルで以下を実行:
      ```bash
      # pandocがあれば
      pandoc income/03-booth-prompts/generated/claude-code-副業プロンプト大全50選.md \
        -o /tmp/claude-code-prompts.pdf --pdf-engine=lualatex \
        -V mainfont="Noto Sans JP"
      ```
    - pandocがなければ、Markdownの内容をGoogleドキュメントに貼り付け → ファイル → PDFでダウンロード
16. サムネイル画像: Canvaで作成（`booth-listing-guide.md` のCanva指示を参照）
    - 手軽にやるなら https://www.canva.com → テンプレート「Instagram投稿」→ ダークネイビー背景に白文字でタイトル入れるだけでOK
17. タグ: `Claude Code`, `AI副業`, `プロンプト`, `ChatGPT`, `副業`, `自動化`
18. 「出品する」をクリック

## 困ったときのFAQ

**Q: BOOTHのPDFが作れない**
A: Markdownファイルの中身をGoogleドキュメントに貼り付け → ファイル → PDFでダウンロード。見た目は後から改善すればいい。まず出品することが最優先。

**Q: noteの有料マガジンの設定が見つからない**
A: マガジン作成時に「定期購読マガジン」を選択する。見つからない場合は、まず無料マガジンを作成 → 後から有料に切り替えることも可能。noteの仕様変更で場所が変わっている場合がある。

**Q: クラウドワークスで良い案件が見つからない**
A: 最初は「タスク形式」の小さい案件（¥1,000〜5,000）でもいい。実績（評価）を1件でも作ることが重要。評価がゼロの状態で高額案件に応募しても通りにくい。

**Q: Amazonアソシエイトの審査に落ちた**
A: サイトの記事が10本以下だと落ちやすい。メディアの記事数が80本以上あるなら普通は通る。落ちた場合はサイトURLの入力ミスを確認。再申請は何度でも可能。

**Q: 全部やる時間がない**
A: 優先順位はこう: **note > BOOTH > クラウドワークス > Amazon**。noteは無料記事の公開だけなら5分で終わる。まずnoteだけやって、残りは明日でもいい。
