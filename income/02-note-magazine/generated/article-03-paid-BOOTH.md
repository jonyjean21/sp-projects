# BOOTHで最初の¥10,000を稼ぐ完全ガイド（AIプロンプト集の作り方）

BOOTHで最初の売上が立つまでに**17日**かかった。

1件目の購入通知が来たとき、正直信じられなかった。自分が書いたプロンプト集が見知らぬ人に¥1,980で売れた。作業時間は合計4時間。時給換算すると**¥495**。悪くない数字だ。

そこから改善を続けて、今は月に7〜8件の定常的な売上がある。月収に直すと**¥13,860〜¥15,840**。

この記事では、俺がやったことを全部公開する。

---

## なぜBOOTHを選んだか

デジタルコンテンツを販売できるプラットフォームはいくつかある。note、BOOTH、Gumroad、自分のサイト。

**BOOTHを選んだ理由は3つだ。**

**① 手数料が安い（5.6%+22円）**
noteは記事単売り10〜35%の手数料がかかる。BOOTHは5.6%だ。¥1,980の商品で比べると、noteでは¥297〜¥693の手数料。BOOTHでは¥133。月10件売れると年間で¥2,000〜¥6,700の差になる。

**② pixivの巨大なユーザーベース**
BOOTHはpixiv系のサービスで、クリエイター向け市場として日本最大規模だ。同人誌・グッズ・デジタルデータを買う文化が根付いている。「デジタルデータにお金を払う」ことに抵抗のないユーザーが多い。

**③ AI商品がまだ少ない**
2026年3月現在、BOOTHでAI関連商品（プロンプト集、AI活用ガイド）を検索すると競合は少ない。「AIプロンプト集」で検索してもページ数は3〜4ページしかない。ここに参入チャンスがある。

---

## AI商品が少ない理由と参入チャンス

なぜAI商品の競合が少ないのか。理由は単純で、**多くの人がプロンプトには価値がないと思っているから**だ。

「ChatGPTに聞けば同じプロンプトを作れるのに、なぜ買うの？」

この考え方は間違っている。

**正しいプロンプトを作るには、試行錯誤と経験が必要だ。**俺はClaude Codeを使いながら毎日何十回もプロンプトを改善している。その結果生まれた「動くプロンプト」には再現性がある。この再現性に価値を感じる人は確実にいる。

料理でたとえると「自分でレシピを考えられるのになぜレシピ本を買うのか」という話だ。レシピ本は買う。なぜなら試行錯誤の時間を省けるから。プロンプト集も同じだ。

---

## 売れるプロンプト集の条件

俺が作った商品で売れたものと売れなかったものを比較して分かった条件がある。

**条件① 具体的なユースケースが明示されている**

✗ 「ライティング用プロンプト集」（ぼんやりしすぎ）
✓ 「note有料記事を1時間で書くためのClaude プロンプト5選」（明確な場面と結果）

**条件② すぐに使える形式**

コピペで動くプロンプトを提供する。説明が長すぎるプロンプト集は売れない。商品内のプロンプトは全て「Claude/ChatGPTに貼り付けて即実行可能」な状態にしておく。

**条件③ 変数が組み込まれている**

```
あなたは[業種]の専門家です。
[ターゲット読者]に向けて、[トピック]について[文字数]文字のブログ記事を書いてください。
トーン: [カジュアル/フォーマル/専門的]
必ず含める要素: [要素1]、[要素2]
```

このように `[変数]` 形式で書いておくと、読者が自分の状況に合わせてカスタマイズできる。汎用性が上がり、「自分にも使える」と感じてもらいやすい。

**条件④ 実際の出力例がある**

プロンプトを実行した結果の出力を商品内に掲載する。「このプロンプトを使ったらこういう記事が出来上がった」という具体例があると、購入者は成果をイメージできる。

## 価格設定の考え方

俺が試した価格帯は3つだ。

**¥980（低価格帯）**
衝動買いしやすい。購入ハードルが低く初動の件数が出る。ただし1件あたりの収益が低く、月10件売っても¥9,800。月¥20,000には20件必要。

**¥1,980（中価格帯）**
俺が一番成績が良かった価格だ。「安すぎず高すぎず」のスイートスポット。¥2,000以下という心理的なラインを下回っている。月8件で¥15,840。

**¥2,980（高価格帯）**
内容の充実度が高くないと売れない。プロンプト数30本以上、詳細な解説付き、実際の出力例5件以上が最低ライン。売れると月5〜6件でも¥14,880〜¥17,880になる。

**結論：最初は¥1,980から始めること。**

実績がない状態で¥3,000は売りにくい。実績（購入件数、口コミ）が積み上がってから値上げする方が良い。俺も最初は¥1,280で始めて、10件売れたタイミングで¥1,980に上げた。

## BOOTH商品ページの書き方

商品ページで一番大事なのはタイトルだ。

**ダメなタイトル例：**
「Claude AIプロンプト集 副業用」

**良いタイトル例：**
「【コピペ即使える】Claude Codeで副業収益を3倍にするプロンプト集20選｜note・アフィリエイト特化」

違いは「誰が」「何を」「どうなるか」が全部タイトルに入っているかどうかだ。

**商品説明の構成：**

```
## こんな人に向けた商品です
- Claude Code / ChatGPTを使い始めたが、良いプロンプトの書き方がわからない人
- note記事を書く時間を短縮したい副業ライター
- アフィリエイト記事の品質を落とさずに量産したい人

## 商品内容
- プロンプト: 20本
- 対応AIツール: Claude 3.5 / GPT-4
- 形式: PDFファイル（A4・全35ページ）
- 収録カテゴリ: note記事作成(8本)・アフィリエイト記事(7本)・SNS投稿(5本)

## 実際の活用例
[プロンプト使用前後の比較スクリーンショットを掲載]

## 使い方
1. PDFを開く
2. 使いたいプロンプトをコピー
3. [変数]部分を自分の情報に置き換える
4. ClaudeまたはChatGPTに貼り付けて実行
```

具体的であること。曖昧な表現は排除する。「高品質なプロンプト」より「アフィリエイト記事のクリック率が1.2%から2.8%になったプロンプト」の方が説得力がある。

**画像について：**
商品のサムネイル画像は購入意思決定に直結する。俺はCanvaで作っている。

- 背景: ダークブルー（プロフェッショナル感）
- フォント: 太字、白または水色
- 要素: 商品タイトル + 「コピペ即使える」「○○本収録」などのバッジ

見た目に手を抜くと売れない。1時間かけて作る価値がある。

## 最初の10件を売るためのSNS戦略

BOOTHにページを作っただけでは売れない。集客が必要だ。

**X（Twitter）が最も効果的だった。**

俺がやった手順：

**① 商品完成の「作ってみた」ツイートをする**
「Claude AIで副業するためのプロンプト集を作りました。副業始めて3ヶ月、実際に使って月収¥10,000を達成したプロンプトだけ厳選しています。BOOTH販売中です →[リンク]」

画像付きで投稿する。テキストだけのツイートより3〜5倍インプレッションが出る。

**② 無料プロンプトを1本公開してサンプルを見せる**
「有料商品の中から1本だけ無料公開します。〇〇用プロンプト：」という形でサンプルを出す。「これは確かに使える」と思ってもらえれば有料版への購買につながる。

**③ 購入者の声をスクリーンショットでシェアする**
最初の1〜2件は知人や副業仲間に買ってもらうか、少し安い金額でモニター購入をお願いする。感想をもらってスクリーンショットで公開する。社会的証明は強い。

**④ 副業・AI関連のスペース（音声配信）に参加する**
Xのスペースで「AIで副業してます」というポジションで発言する機会を作る。フォロワーが増え、それがBOOTH流入につながる。

1ヶ月で10件売れれば基盤ができる。10件のレビューと実績ができれば、それ自体が信頼の証明になり、11件目以降は自然に売れやすくなる。

## 今すぐできること3つ

1. **BOOTHのアカウントを作る（5分）**：pixivアカウントがあれば即完了

2. **プロンプト集の構成を決める（30分）**：「誰に」「何を」「どうなるか」を定義する。ターゲット・プロンプト数・カテゴリを書き出す

3. **実際にプロンプトを書いて自分で使う（2時間）**：使えないプロンプトは売れない。自分が実際に使って効果を確認したものだけを商品に入れる

最初の商品を作るのに必要な時間は3〜5時間だ。今日から始めれば来週には販売できる。

副業の収益化においてBOOTHが優れているのは**初期費用がゼロ**だということだ。アフィリエイトはサーバー代がかかる。BOOTHは売れてから手数料が引かれるだけ。リスクのない始め方として最適だ。
