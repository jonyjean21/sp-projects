### プロンプト 1: 記事構成（アウトライン）自動生成

**用途**: キーワードからSEOに強い記事の骨格を作る
**使い方**: 記事を書き始める前に必ずこれを先に実行する

```

検索意図: {{購買検討中 / 情報収集中 / 比較検討中}}

出力形式:
- タイトル案（3パターン、各40文字以内）
- メタディスクリプション（120文字）
- H2見出し（5〜7個）
- 各H2の下にH3見出し（2〜3個ずつ）
- 推奨文字数
- 入れるべきキーワードのバリエーション（5個）
```

**コツ**: 検索意図を「購買検討中」にすると商品紹介・アフィリエイト向けの構成になる
