## 🤖 自動化の仕組み

```
毎日 cron で以下を実行:
├── 09:00 X自動投稿Bot (アフィリエイト + 情報発信)
├── 10:00 SEO記事自動生成 (MOLKKY HUB / アフィリエイトブログ)
├── 11:00 note記事自動下書き生成
├── 18:00 X自動投稿Bot (夜の部)
└── 23:00 翌日コンテンツ準備 + パフォーマンスレポート生成

毎週 1回 (日曜):
├── アフィリエイト収益チェック + CSV記録
├── キーワードリサーチ自動実行
└── 翌週コンテンツカレンダー自動生成

毎月 1回:
└── 月次収益レポート自動生成
```
