## 提案文本文

はじめまして。AI × WordPress自動化が得意なエンジニア「SP」と申します。

「商品数が多くて説明文を書く時間がない」「コピペ作業で登録ミスが頻発する」──Claude APIとWooCommerce REST APIを組み合わせれば、商品CSVをアップロードするだけで説明文の生成から登録まで全自動化できます。

私はClaude Codeを使ったSEO記事量産システムを自社メディア（月3,000セッション）に導入しており、1記事3分以内での高品質コンテンツ生成を実現しています。同じアプローチで商品説明文の一括生成・WooCommerce自動登録スクリプトを構築します。

【実現方法】
商品名・スペック・カテゴリをまとめたCSVをインポート→PythonスクリプトがClaude APIに商品情報を渡し、SEO最適化された説明文を生成→WooCommerce REST APIで商品情報を一括更新。生成文のトーン（カジュアル/フォーマル）や文字数もパラメータで制御可能です。

予算3万円で100商品までの一括処理スクリプトを納品します。納期は5〜7営業日。スクリプトはご自身で再利用できる形式で渡します。

まず「商品数」と「現在の商品データの形式（CSV/Excel等）」を教えていただけますか？すぐに見積もりします。
