■ 主な実績

【モルック情報メディアの全自動運営】
スポーツ情報メディアの記事作成・大会情報収集・SNS投稿を全自動化。Claude Codeを使ったSEO記事量産システムにより月3,000セッションを達成しており、運営にかかる手作業時間をほぼゼロに削減しています。

【GAS × Firebase 大会情報自動収集システム】
複数サイトから大会情報をスクレイピング→Firebase Realtime Databaseに保存→Webサイトへ自動反映する仕組みを構築。手動更新が不要になり情報鮮度が大幅に向上しました。

【Claude API × LINE Bot 顧客対応自動化】
FAQデータを読み込ませたClaude API連携LINE Botを構築。文脈を理解した自然な自動返信と、対応不能時のエスカレーション通知を実装しました。
