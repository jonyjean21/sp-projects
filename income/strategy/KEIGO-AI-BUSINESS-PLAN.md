### ChatGPTは脅威か？

脅威だが致命的ではない。理由:
1. ビジネス敬語の生成はできるが「これが正しいかどうか判断できない」- 外国人ユーザーは出力を検証できない
2. プロンプトエンジニアリングが必要 → 平均ユーザーはできない
3. Gmail/Outlookとの統合がない → 毎回コピペが必要
4. ビジネス日本語の文化的文脈の説明がない

**差別化戦略**: ChatGPTより「ビジネス日本語専用」「ワンクリック」「なぜその表現が正しいかの解説付き」

## 3. プロダクト仕様

### コア機能（MVP v0.1 - 2週間で完成）

```
INPUT:  英語または日本語（自分の言いたいことを書く）
        ↓
OUTPUT: ① 修正済みビジネス日本語
        ② ローマ字読み
        ③ 修正箇所の英語解説（なぜその表現が正しいか）
        ④ フォーマリティレベル（Formal/Semi-formal etc）
        ⑤ 文化的Tips（「日本のビジネスでは〜が重要」）
```

### 差別化ポイント

1. **英語→日本語フロー** (誰もやっていない): 英語で意図を書けば敬語メールに変換
2. **解説付き**: 「なぜこの表現が正しいか」を英語で説明 → 学習効果がある
3. **シナリオ別テンプレート**: 断るメール/謝罪メール/依頼メール etc
4. **ビジネス文化ガイド**: 敬語レベル選択の指針
5. **Chromeエクステンション**: Gmail上でワンクリック修正（将来）

### 技術スタック（コスト: 初月¥0）

```
Frontend: Next.js 14 (App Router)
Hosting:  Vercel (無料枠)
Auth:     Clerk (無料枠: 10,000 MAU)
Payment:  Stripe (サブスクリプション)
DB:       Supabase (無料枠: 500MB)
AI:       Claude Haiku (1修正あたり約¥0.4)
Email:    Resend (無料枠: 3,000通/月)
```

**API コスト試算**:
- 1修正あたり: 約$0.003（¥0.4）
- $29/月のユーザーが100回修正: API費用 = $0.30 → **粗利率 99%**
- $199/月のチームが500回修正: API費用 = $1.50 → **粗利率 99.2%**
