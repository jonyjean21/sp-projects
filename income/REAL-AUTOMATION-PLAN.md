## Layer 1: クラウドワークス × AI（最速）

### なぜ最速か
- 受注1件で¥30,000〜100,000
- 提案文はAIが生成済み
- 納品物（記事・GAS・API連携）もAIが作れる
- 実績0でも「Claude Codeで自動化します」が差別化になる

### 自動化フロー
```
毎日9時: CWスクレイパーが新着案件を取得
  → Claude API で案件スコアリング（合致度）
  → スコア高い案件の提案文を自動生成
  → ファイルに保存（SPが確認 → コピペで応募）
  ※ 確認は1日5分
```

## Layer 2b: note 有料マガジン

### 成功パターン（リサーチ結果）
- AI下書き + 実体験 = 最強
- 「AIのみ」は読まれない（2025年以降Google・読者共に評価低下）
- **SPの実体験が武器**: Claude Codeで実際に稼いでいる当事者

### マガジン設計
```
マガジン名: 「Claude Code × 副業 リアルレポート」
価格: ¥980/月
記事: 週1〜2本（AI下書き → SP実体験追記 → publish）

記事テーマ（3ヶ月分）:
  Week 1: MOLKKY HUBをAIで全自動運営している話
  Week 2: クラウドワークスでAI納品した実録
  Week 3: 3月の収益公開（正直レポート）
  Week 4: Claude Code vs ChatGPT、副業で使い倒した比較
  ... (AIが毎週テーマを提案)
```

### 自動化フロー
```
毎週月曜: AI が記事下書きを生成 → ファイル保存
SPが実体験を追記 (15分)
毎週水曜: publishして告知文をBufferに投稿
```

## 🔧 技術スタック（変更後）

```
言語: Node.js (ESM)
AI: Anthropic Claude API (claude-haiku-4-5: 安い・速い)
    └ 生成コンテンツ用: haiku（コスト削減）
    └ 提案文・重要コンテンツ: sonnet（品質重視）
CMS: WordPress REST API (MOLKKY HUB)
スケジューラー: cron (Mac) or GitHub Actions
X投稿: Buffer無料プラン（APIなし）
PDF: pandoc または marked (node)
データ: JSON file-based（DBなし・シンプル）
```
