### 5-2. 受付フロー（効率化のポイント）
```
来場 → 受付（支払い・整理券発行）→ 待機エリア → 搭乗エリア → フライト → 降機 → 写真DLエリア → 出口
```

**プロのテクニック:**
1. **整理券方式**: 到着順に番号札を配布。大体の搭乗時刻を案内し、その間自由に過ごしてもらう
2. **決済の高速化**: 現金+QR決済（PayPay等）の2レーン
3. **待ち時間コンテンツ**: 気球の仕組み解説パネル、フォトスポット、飲食エリアで退屈させない
4. **搭乗記念撮影**: 搭乗直前にスタッフが撮影 → QRコードで後日DL（リスト取得にもなる）

## 6. 成功事例から学ぶ「差がつく施策」

### 6-1. 佐賀インターナショナルバルーンフェスタに学ぶ
日本最大の気球イベント。100万人以上が来場。
- **学び**: 早朝の非日常感が最大の魅力。日の出 × 気球の写真は確実に映える
- **取り入れる**: 日の出タイミングのフライトを「ゴールデンタイム」として演出。SNS投稿を促す仕掛け

### 6-2. 渡良瀬バルーンレースに学ぶ
地域密着型の小規模気球イベント。
- **学び**: 地元の飲食・物産との連携で「気球 + α」の体験を作る
- **取り入れる**: キッチンカーだけでなく、地元の店や団体に声をかけ、ミニマルシェ的な要素を加える

### 6-3. コミュニティイベントの成功パターン
- **「文化祭効果」**: スタッフ自身が楽しんでいるイベントは来場者も楽しい。スタッフ参加費モデルはこれを自然に実現している
- **「役割の明確化」**: 全員が何をすべきか迷わない状態が最重要。事前のブリーフィング資料で具体的に示す
- **「終わりの演出」**: 片付け → 打ち上げの流れが、次もやりたいという気持ちを生む

### 6-4. SP × MACHAP ならではの付加価値
1. **デジタルチケット/予約システム**: SPが構築可能。紙のやりとりを最小化
2. **リアルタイム待ち時間表示**: 簡易Webページで現在の待ち人数を表示
3. **イベント記録の自動化**: 写真整理・収支集計を仕組み化
4. **次回への導線**: 参加者にLINE登録を促し、次回案内を自動送信

## 7. リスク管理

| リスク | 確率 | 影響 | 対策 |
|--------|------|------|------|
| 荒天中止 | 中 | 高 | 予備日設定 or キャンセルポリシー明確化 |
| 来場者不足 | 低 | 中 | 早めの告知開始、複数チャネルでの集客 |
| 来場者過多 | 低 | 中 | 整理券制で上限管理、次回案内で期待つなぎ |
| スタッフ不足 | 中 | 高 | 最低人数ラインを設定、早めの募集 |
| 機材トラブル | 低 | 高 | 業者との事前確認、代替プログラム準備 |
| 事故・怪我 | 極低 | 極高 | 保険加入、安全マニュアル、救護体制 |
| 近隣トラブル | 低 | 中 | 事前挨拶、騒音配慮（早朝のため特に注意） |
