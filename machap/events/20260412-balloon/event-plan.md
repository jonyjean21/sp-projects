## 4. 収支シミュレーション（2機運用 / 500人来場）

### キャパシティ計算（重要）
```
2機 × 7人/回 × 6回/時間（乗降込み10分サイクル） = 84人/時間
3時間稼働（06:00-09:00） = 252人（理論最大）
実稼働率80% = 約200人が搭乗可能

500人来場のうち搭乗希望者がどれだけいるか = 要予約管理
→ 予約制で搭乗枠200人を事前に埋める
→ 残り300人は見学 + キッチンカー + イベント参加
```

**⚠️ 課題: 500人来場に対し搭乗キャパは最大200人**
→ 予約制で搭乗枠を明確にし、「見学無料 + 搭乗は予約者のみ」の設計が必要

### 収入
| 項目 | 単価 | 数量 | 金額 | 備考 |
|------|------|------|------|------|
| 大人乗船料 | ¥3,500 | 150名 | ¥525,000 | 搭乗200枠のうち大人150 |
| 子供乗船料（一般） | ¥1,500 | 30名 | ¥45,000 | PTA関連以外の子供 |
| 小学生（PTA関連） | ¥0〜割引 | 20名 | ¥0〜 | PTA予算で補填 |
| スタッフ参加費 | ¥10,000 | 20名 | ¥200,000 | ¥5,000の場合は¥100,000 |
| **収入合計** | | | **¥770,000** | スタッフ¥10,000の場合 |

### 支出
| 項目 | 金額 | 備考 |
|------|------|------|
| 気球外注（2機） | ¥1,100,000 | 株式会社気球屋 / 中さん立替→イベント後精算 |
| 保険 | 主催者負担 | |
| 会場費 | 主催者負担 | 多田東小学校 |
| キッチンカー | 主催者負担 | 3台手配済み |
| 備品・消耗品 | ¥20,000 | テープ、看板、受付用品等 |
| **支出合計（こちら負担分）** | **¥1,120,000** | |

### 損益
```
収入 ¥770,000 - 支出 ¥1,120,000 = -¥350,000

ただし:
- PTA予算からの補填額が不明（→ 中さんに確認必要）
- 乗船料の正確な金額を中さんがまだ把握していない
- 主催者側の予算構造（地域予算、PTA予算等）の全体像が見えていない

→ 次回打ち合わせ（~3/27）で収支の全体像を確認すべき
```

**⚠️ 要確認事項:**
1. PTA予算の具体的な金額と使い道
2. 大人乗船料の正確な金額
3. 主催者側の収支バランス（こちらとの精算方法）
4. 気球代110万円の精算タイミングと方法

| 安全管理 | 2 | スタッフ | 搭乗エリア周辺の安全確保・子供の安全 |
| 会場案内 | 2 | スタッフ | 駐車場案内（スタッフ車）、来場者誘導、トイレ案内 |
| 記録 | 1 | スタッフ | 写真・動画撮影 |
| 遊軍 | 4+ | スタッフ | 交代要員、キッチンカー連携、急な対応 |

### 受付フロー（500人対応版）
```
【予約者】
来場 → 受付（予約番号確認・現金支払い）→ 整理券受取
→ 搭乗時間まで自由（キッチンカー・見学）→ 搭乗 → 降機

【見学者・予約なし】
来場 → 案内（搭乗は予約者のみと案内）→ 自由に見学・飲食

【受付効率化】
- 受付3レーン（500人を3時間で捌く = 1レーンあたり56人/時間 = 約1分/人）
- 予約番号制で確認を高速化
```
